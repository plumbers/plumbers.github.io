[Грани Агни Йоги 1959г](/grani_agni/1959)

Учителем Советы даются для применения их в [жизни](/tags/жизнь). Поэтому каждый Совет следует применить по мере [сил](/tags/сила) и разумения. Углубим [понимание](/tags/понимание) рефлекторности [мышления](/tags/мышление). Оно означает, прежде всего, автоматизм [мыслей](/tags/учитель) Приоткрывает Завесу.
