[Грани Агни Йоги 1958г](/grani_agni/1958)

Пусть трудные времена явятся благословенными для любящего всем сердцем Владыку. Трудность есть нечто, заряженное [энергией](/tags/энергия) противодействия, сопротивления или инертности. Преодоление их дает эту силу в [руки](/tags/путь).
