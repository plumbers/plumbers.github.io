[Грани Агни Йоги 1958г](/grani_agni/1958)

[Руку](/tags/рука) Водящую усмотрите в том, что каждодневно импульс дается и [силы](/tags/сила) для дальнейшего продвижения. Дается именно то, что в данный момент надо для [духа](/tags/Дух). Это и есть хлеб наш насущный, пища для [духа](/tags/Дух), без которой восхождение его невозможно. Пищей лишь только [земной](/tags/Земля) [духа](/tags/Дух) насытить нельзя. Надо приучить себя к этому [устремлению](/tags/устремление) каждодневному, чтобы сердцем открытым воспринимать пищу текущего дня. [Луч](/tags/космос) Правом.
