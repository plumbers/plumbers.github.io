[Грани Агни Йоги 1971г](/grani_agni/1971)

Теперь уже нельзя допустить, чтобы тени прошлого заслонили горизонт [будущего](/tags/будущее). Утверждается крепко господство [воли](/tags/воля) над всеми попытками астрала погрузить [сознание](/tags/сознание) в то, чем когда-то он жил. «В прошлом все сожжено для Огненной Йоги».
