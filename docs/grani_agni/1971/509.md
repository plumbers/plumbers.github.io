[Грани Агни Йоги 1971г](/grani_agni/1971)

___509___
Непоколебимое доверие к [Учителю](/tag/#учитель) открывает [пути](/tag/#путь) для новых нахождений. Недоверие и сомнение их закрывают. Рост [духа](/tag/#Дух) идет незаметно для глаза, как рост травы. Но результаты его явны, по крайней мере, для нас, видящих изменения, происходящие в ауре ученика. То огромное количество сведений о [вещах](/tag/#вещь), относящихся к Учению, и [мыслей](/tag/#мысль), сопутствующих им, является посевом зерен Света на ниве [сознания](/tag/#сознание). Этот посев и будущие всходы остаются с учеником навсегда. Потому будьте в вере покойны.
