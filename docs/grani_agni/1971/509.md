[Грани Агни Йоги 1971г](https://127.0.0.1:4002/grani_agni/1971)

___509___
Непоколебимое доверие к [Учителю](../../../tags/#учитель) открывает [пути](../../../tags/#путь) для новых нахождений. Недоверие и сомнение их закрывают. Рост [духа](../../../tags/#Дух) идет незаметно для глаза, как рост травы. Но результаты его явны, по крайней мере, для нас, видящих изменения, происходящие в ауре ученика. То огромное количество сведений о [вещах](../../../tags/#вещь), относящихся к Учению, и [мыслей](../../../tags/#мысль), сопутствующих им, является посевом зерен Света на ниве [сознания](../../../tags/#сознание). Этот посев и будущие всходы остаются с учеником навсегда. Потому будьте в вере покойны.
