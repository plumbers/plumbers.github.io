[Грани Агни Йоги 1971г](/grani_agni/1971)

Цель ученика – научиться управлять, или владеть, своими чувствами. Это означает – держать [мысли](/tags/[мысль](/tags/мысль)) под контролем. Только мыслью можно управлять эмоциями. Они текут автоматически, то есть рефлекторно. Лунную рефлекторность [сознания](/tags/сознание) только огненной [волей](/tags/воля) можно пресечь.
