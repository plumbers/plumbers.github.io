[Грани Агни Йоги 1971г](/grani_agni/1971)

___392___
От строителя Нового Мира требуется [понимание](/tag/#понимание) существования обоих миров, иначе строение будет однобоким, то есть несовершенным. При одностороннем [понимании](/tag/#понимание) мира ошибки неизбежны. [Сознание](/tag/#сознание) строителя нуждается в перерождении, то есть в объединении в нем обоих миров.
