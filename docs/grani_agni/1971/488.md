[Грани Агни Йоги 1971г](/grani_agni/1971)

Ученик должен лечить себя сам с помощью своей психической [энергии](/tags/энергия), то есть заботиться о поддержании [равновесия](/tags/равновесие) или [гармонии](/tags/гармония) в своем организме. Зачаток каждого заболевания, то есть нарушение этой [гармонии](/tags/гармония), должен он восстанавливать [возможно](/tags/возможности) направлениям.
