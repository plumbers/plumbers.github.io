[Грани Агни Йоги 1971г](/grani_agni/1971)

При полном объединении сознаний наступает полное взаимопонимание. Такое взаимопонимание быть может только сердечным. Слишком мало [места](/tags/место) отводится [сердцу](/tags/Дух), сужденным ему от начала времен.
