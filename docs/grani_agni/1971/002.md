[Грани Агни Йоги 1971г](/grani_agni/1971)

___2___
Конец Кали Юги будет отмечен небывалым нагнетением кармических [условий](/tag/#условие) как в общепланетном масштабе, так и в личной [жизни](/tag/#жизнь) каждого [человека](/tag/#человек). Предстоит последняя уплата по счетам за прошлую задолженность. Не может войти в [будущее](/tag/#будущее) человек, обремененный долгами. Сатиа Юга не имеет утешения для должников. Остаться за порогом Нового Мира – это самое страшное, что может ожидать [человека](/tag/#человек). Там останутся сознательные служители тьмы. Но каждый войдет, в ком есть хоть искорка Света. Время конечных счетов определит судьбу каждого [человека](/tag/#человек). Грозное время идет [решения](/tag/#решение) судеб [человеческих](/tag/#человечество).
