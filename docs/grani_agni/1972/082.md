[Грани Агни Йоги 1972г](https://127.0.0.1:4002/grani_agni/1972)

___82___
Для чего испытания, трудности, преодолевание своей низшей природы? Только для того, чтобы, очистившись, подойти еще ближе к Владыке. Каждый ученик для Него тягость являет и обременение, вызванные огромной разницей в излучениях [аур](../../../tags/#аура). Значит, свои излучения каждый хотящий стать как можно ближе к Владыке должен поднять до требуемой высоты.
