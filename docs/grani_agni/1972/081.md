[Грани Агни Йоги 1972г](/grani_agni/1972)

[Сознание](/tags/сознание) [человеческое](/tags/человечество) и его психические продукты есть явления тонкого порядка. Нельзя их ни взвесить, ни измерить, ни приложить к ним метод обычного анализа. [Мысль](/tags/мысль) будет изучаться лабораторно-научно. Поймут тогда отрицатели ее скрытую силу, которая может вырастать в своей мощи в зависимости от числа сгармонизированных сознаний.
