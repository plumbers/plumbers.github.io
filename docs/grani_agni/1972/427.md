[Грани Агни Йоги 1972г](/grani_agni/1972)

Сознательное усовершенствование [человеческого](/tags/человечество) аппарата должно быть первейшей обязанностью [человека](/tags/человек), ибо только этим [путем](/tags/путь) недостижимое становится достижимым. [Аппарат](/tags/аппарат) этот сложен необычайно. Он семеричен. Начиная с плотного [тела](/tags/служение) Иерархии Света.
