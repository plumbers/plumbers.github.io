[Грани Агни Йоги 1963г](/grani_agni/1963)

___572___
Те, кто продолжает [дело](/tag/#дел) мое, не признают интуитивно, что со смертью моего [тела](/tag/#тело) закончил существование мой [дух](/tag/#Дух). Служа моему делу, служат моему духу, устанавливая с ним прочные связи. Конечно, признание меня, даже посмертно, есть утверждение огненной действительности, вопреки видимости плотного мира. Реальность Незримого Высшего Мира может утверждать только [дух](/tag/#Дух), готовый к тому, чтобы идти по [Пути](/tag/#путь) Света. Их немало, готовых, в каждом усмотрим воина Света.
