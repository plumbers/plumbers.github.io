[Грани Агни Йоги 1963г](https://127.0.0.1:4002/grani_agni/1963)

___572___
Те, кто продолжает [дело](../../../tags/#дел) мое, не признают интуитивно, что со смертью моего [тела](../../../tags/#тело) закончил существование мой [дух](../../../tags/#Дух). Служа моему делу, служат моему духу, устанавливая с ним прочные связи. Конечно, признание меня, даже посмертно, есть утверждение огненной действительности, вопреки видимости плотного мира. Реальность Незримого Высшего Мира может утверждать только [дух](../../../tags/#Дух), готовый к тому, чтобы идти по [Пути](../../../tags/#путь) Света. Их немало, готовых, в каждом усмотрим воина Света.
