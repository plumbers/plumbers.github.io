[Грани Агни Йоги 1963г](/grani_agni/1963)

Сын Мой, личность дана [человеку](/tags/человек) для выполнения определенной задачи. Эта задача заключается в том, чтобы собрать на [Земле](/tags/Земля) материал, необходимый для сознательного выявления его Бессмертной Индивидуальности на всех планах бытия. Личность живет самой [напряженной](/tags/напряжение) [жизнью](/tags/жизнь) на плане [земном](/tags/Земля). С одной стороны, она есть следствие всех прошлых личностей, подобно бусам, нанизанным на стержень [духа](/tags/человек).
