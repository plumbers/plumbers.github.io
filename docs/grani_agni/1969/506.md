[Грани Агни Йоги 1969г](/grani_agni/1969)

___506___
Чувствознание определяет, кому и когда можно дать полученные [знания](/tag/#знание). Именно чувствознание дает [возможность](/tag/#возможности) говорить по [сознанию](/tag/#сознание). С одним хочется поделиться широко, с другим [желания](/tag/#желание) нет, хотя и тот и другой знакомы с Учением. Что-то внутри безошибочно определяет. Это и есть уже [знание](/tag/#знание) [духа](/tag/#Дух).
