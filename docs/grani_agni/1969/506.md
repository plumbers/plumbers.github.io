[Грани Агни Йоги 1969г](/grani_agni/1969)

Чувствознание определяет, кому и когда можно дать полученные [знания](/tags/знание). Именно чувствознание дает [возможность](/tags/возможности) говорить по [сознанию](/tags/сознание). С одним хочется поделиться широко, с другим [желания](/tags/желание) нет, хотя и тот и другой знакомы с Учением. Что-то внутри безошибочно определяет. Это и есть уже [знание](/tags/знание) [духа](/tags/Дух).
