[Грани Агни Йоги 1969г](/grani_agni/1969)

[Человеческий](/tags/человечество) организм есть лаборатория для всех ощущений, которые с помощью [воли](/tags/человечество) организма из отрицательных в положительные. Много Света можно вносить в окружающую среду умением настраивать свой микрокосм на желаемой волне Света.
