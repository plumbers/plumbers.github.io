[Грани Агни Йоги 1969г](/grani_agni/1969)

Теперь понимаете, для чего нужно утверждение имени Гуру, для чего нужны его картины и [книги](/tags/книга) и почему известность его, столь широко распространенная в мире, нужна не для его прославления, но для утверждения [Учения](/tags/учение) Владык. Возвеличивая его, возвеличиваете Пославших его на [подвиг](/tags/подвиг). Ведь не кончилось ничто, а наоборот, [будущее](/tags/будущее), сужденное [человечеству](/tags/мир) миссии. Плоды созревают ко времени. И будет что собирать. И хватит на всех, ибо посев был великим.
