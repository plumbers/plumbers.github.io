[Грани Агни Йоги 1967г](/grani_agni/1967)

[Мысль](/tags/[мысль](/tags/мысль)) о Владыке будет мыслью о Свете, ибо [Свет](/tags/свет) в Нем. Не могут приближаться темные [сущности](/tags/сущность) к [сознанию](/tags/сознание), наполненному Владыкой. [Свет](/tags/свет) Лика Его отбрасывает их, словно вихрь листья сухие. Мощь воздействия Лика следует [знать](/tags/познание). Конечно, наполнение должно быть достаточно полным, чтобы [места](/tags/место) уже не оставалось для чего-либо другого. Двойственность [мыслей](/tags/мысль) разрушительна. Монолитное [мышление](/tags/мышление) являет собою силу.
