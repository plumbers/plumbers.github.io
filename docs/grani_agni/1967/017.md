[Грани Агни Йоги 1967г](/grani_agni/1967)

[Мысль](/tags/[мысль](/tags/мысль)), заряженная чувством, то есть [энергией](/tags/энергия) [сердца](/tags/сердце), соединяет [сознание](/tags/сознание) со сферами, созвучными [мысли](/tags/[мысль](/tags/мысль)). Соблюсти [мысль](/tags/мысль) будет прямым [путем](/tags/путь) приближения к [Учителю](/tags/учитель) Света.
