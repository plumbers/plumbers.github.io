[Грани Агни Йоги 1953г](/grani_agni/1953)

Через ещё большее [спокойствие](/tags/спокойствие) приближаешься. Оно венчает чело победителя. Потому и названо оно венцом [духа](/tags/Дух). Правильно. Испытания даются для утверждения нужных [качеств](/tags/луч) Мой, волны Света несущий, станет светом твоим, являемым сущностью твоего микрокосма в мире вовне.
