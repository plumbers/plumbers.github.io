[Грани Агни Йоги 1953г](https://127.0.0.1:4002/grani_agni/1953)

___361___
Утверждаю мощь [действия](../../../tags/#действие), когда в [действии](../../../tags/#действие) собрана вся сила [духа](../../../tags/#Дух). Собрать [силы](../../../tags/#сила) и ринуть их по одному направлению будет знаком успеха. Распылять свои [силы](../../../tags/#сила) по мелочам и по пустякам каждый умеет. Но собирателей [сил](../../../tags/#сила) искусных нет.
