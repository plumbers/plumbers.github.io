Без Иерархии [дух](/tags/Дух) не может творить. Творчество есть такая степень [напряжения](/tags/напряжение) [сознания](/tags/понимание) искусства, Утверждаем творчество всенародное как новую ступень эволюции. Утверждаем творчество для всех.
