[Грани Агни Йоги 1966г](/grani_agni/1966)

Иногда очень полезно подумать о том, насколько нам не нужны все те нагромождения, которые наслаиваются вокруг личности тех, кого мы позвали за собою. [Учитель](/tags/учитель) Указует собрать в [путь](/tags/путь) дальний только ценные [вещи](/tags/вещь), только те, которые взять можно с собою за пределы плотного мира, за пределы астрального, в Высшие Сферы Бессмертной Триады. Все остальное будет излишней поклажей, которая отягощает подъем.
