[Грани Агни Йоги 1966г](/grani_agni/1966)

Без соответствия нет созвучия. Соответствие необходимо, чтобы созвучие состоялось. Утверждение [качеств](/tags/качество) [духа](/tags/Дух) нужно для того, чтобы было чем соответствовать. Созвучат вибрациям [ауры](/tags/аура) [Учителя](/tags/учитель) элементы соответствия. [Качества](/tags/качество) [духа](/tags/Дух) и являются этими элементами, ибо состоят из них. [Качества](/tags/качество) [духа](/tags/Дух) состоят из элементов [огня](/tags/огонь). Огнями своей [ауры](/tags/аура) созвучит человек с аурой Иерарха. И становится человек огненно звучащим. [Закон](/tags/закон) созвучия управляет миром во всех сферах. Элементы созвучия можно собирать сознательно. Антиподом созвучия будет диссонанс, дисгармония. Тьма характерна отсутствием [гармонии](/tags/гармония). [Сущность](/tags/сущность) явления ее разрушительна. Итак, созвучие, соответствие, [гармония](/tags/гармония) – основы светоносности. Явления Света гармоничны. [Спокойствие](/tags/спокойствие) и [равновесие](/tags/равновесие) есть [гармония](/tags/гармония) центров. [Спокойствие](/tags/спокойствие) и [равновесие](/tags/равновесие) обуславливают глубину созвучия. Симфония [качеств](/tags/качество) делает его полнострунным. Огненное звучание [возможно](/tags/возможности) только при наличии утверждаемых [качеств](/tags/качество). По степени накопления [качеств](/tags/качество) – и созвучие.
