[Грани Агни Йоги 1966г](/grani_agni/1966)

___107___
Соответствие – основной [закон](/tag/#закон) Надземного пребывания. По тому, что внутри, легко можно определить, что будет вовне, в окружении тонком. Принцип созвучия обуславливает притяжение родственных элементов, создающих содержание тонких восприятий. Каждый имеет по себе. На это может рассчитывать человек, всю свою жизнь на [Земле](/tag/#Земля) пребывавший в [мыслях](/tag/#[мысль](/tag/#мысль)) нечистых и злобных и поступавший соответственно с ними. Соблюсти [мысль](/tag/#мысль) будет условием, необходимым для создания для своего [духа](/tag/#Дух) достойного окружения, ибо мыслью творится [будущее](/tag/#будущее) [человека](/tag/#человек) в астрале и выше.
