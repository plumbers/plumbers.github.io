[Грани Агни Йоги 1966г](/grani_agni/1966)

___58___
Ларец для тех, кто, пройдя через горнило [земных](/tag/#Земля) испытаний, сумеет удержать Серебряную Нить связи с Учителем. Испытания на прочность связи одни из труднейших. Для кого-то это были костры, пытки, растерзание зверями, для кого-то одиночество и лишение привычных [условий](/tag/#условие), для кого-то богатство и власть. Для каждого свое, но не менее трудное, чем для другого. И выдержавшие прошли через все, не порвав связей [духа](/tag/#Дух). Удержать Серебряную Нить много труднее, нежели ее установить. Об этом следует [думать](/tag/#дума), ибо [земная](/tag/#Земля) жизнь есть проверка стойкости [духа](/tag/#Дух), подготовка его для [жизни](/tag/#жизнь) в Надземном.
