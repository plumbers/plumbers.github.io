[Грани Агни Йоги 1966г](/grani_agni/1966)

Изображение есть видимый посредник для Общения с Тем, Кто запечатлен на Изображении. Даже обычная фотокарточка при смотрении на нее соединяет смотрящего с оригиналом. Для утверждения незримых нитей связи нужны зримые [предметы](/tags/предмет). Именно [предметы](/tags/предмет), так как, помимо изображений, и они могут способствовать контакту.
