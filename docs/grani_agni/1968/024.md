[Грани Агни Йоги 1968г](/grani_agni/1968)

Некоторые положения древних писаний можно назвать огненными формулами, утверждающими Законы [жизни](/tags/жизнь). Например: «По вере вашей дано будет вам»; «по вере», говоря другими словами, по [сознанию](/tags/сознание), – эта формула останется навсегда законом для того, кто вступает в [Мир](/tags/значение) и где она руководит.
