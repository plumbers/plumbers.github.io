[Грани Агни Йоги 1968г](https://127.0.0.1:4002/grani_agni/1968)

___544___

[Путь](../../../tags/#путь) указуется по вершинам. По подъемам [сознания](../../../tags/#сознание) отмечаются его вехи. По взлетам [мысли](../../../tags/#мысль) прокладываются направления устремленного [духа](../../../tags/#Дух) в [пространстве](../../../tags/#пространство). [Путь](../../../tags/#путь) пролегает поверх серой повседневности. Поднятые головы ищут [Луч](../../../tags/#луч). Но даже самолет не взлетит, если не подготовлено поле для взлета. Поле [сознания](../../../tags/#сознание) тоже требует подготовки, чтобы [дух](../../../tags/#Дух) мог подняться и взлететь над пучиною [жизни](../../../tags/#жизнь).
