[Грани Агни Йоги 1968г](/grani_agni/1968)

___544___
[Путь](/tag/#путь) указуется по вершинам. По подъемам [сознания](/tag/#сознание) отмечаются его вехи. По взлетам [мысли](/tag/#мысль) прокладываются направления устремленного [духа](/tag/#Дух) в [пространстве](/tag/#пространство). [Путь](/tag/#путь) пролегает поверх серой повседневности. Поднятые головы ищут [Луч](/tag/#луч). Но даже самолет не взлетит, если не подготовлено поле для взлета. Поле [сознания](/tag/#сознание) тоже требует подготовки, чтобы [дух](/tag/#Дух) мог подняться и взлететь над пучиною [жизни](/tag/#жизнь).
