[Грани Агни Йоги 1954г](/grani_agni/1954)


Человек, поднявший [руку](/tags/рука), должен ее опустить, раскрывши рот, должен его закрыть, вошедший в воду, должен из нее выйти, заговоривший должен замолчать, уявивший [действие](/tags/действие), должен его закончить, то есть, выведя свой микрокосм из определенного [состояния](/tags/космос).
