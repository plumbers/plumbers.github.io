[Грани Агни Йоги 1951г](/grani_agni/1951)

___317___
[Огонь](/tag/#огонь) поругаем не бывает. Думающие служить двум господам не пройдут в Шамбалу. Нельзя строить свое [равновесие](/tag/#равновесие) ни на каких внешних обстоятельствах. Они непрочны, значит, и не годны для [равновесия](/tag/#равновесие). Только в своем вечном духе его, то есть [равновесия](/tag/#равновесие), фундамент.
