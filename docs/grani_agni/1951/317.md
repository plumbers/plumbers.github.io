[Грани Агни Йоги 1951г](https://127.0.0.1:4002/grani_agni/1951)

___317___

[Огонь](../../../tags/#огонь) поругаем не бывает. Думающие служить двум господам не пройдут в Шамбалу. Нельзя строить свое [равновесие](../../../tags/#равновесие) ни на каких внешних обстоятельствах. Они непрочны, значит, и не годны для [равновесия](../../../tags/#равновесие). Только в своем вечном духе его, то есть [равновесия](../../../tags/#равновесие), фундамент.
