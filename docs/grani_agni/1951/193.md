[Грани Агни Йоги 1951г](https://127.0.0.1:4002/grani_agni/1951)

___193___

Нам нужно создать совершенно новых [людей](../../../tags/#люди). Вот два [человека](../../../tags/#человек), воспринимающих [огонь](../../../tags/#огонь) ([луч](../../../tags/#луч)): оба его имеют, но один умеет им пользоваться, а другой – нет. Надо [учиться](../../../tags/#учение) владеть огнями. Овладеть, для того чтобы уметь применять в [жизни](../../../tags/#жизнь). Для [жизни](../../../tags/#жизнь) дается огненное могущество. Ясность Лика [Учителя](../../../tags/#учитель) ставит нас с Ним в непосредственное соприкосновение. Углубляйте ясность.
