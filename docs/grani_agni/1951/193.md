[Грани Агни Йоги 1951г](/grani_agni/1951)

___193___
Нам нужно создать совершенно новых [людей](/tag/#люди). Вот два [человека](/tag/#человек), воспринимающих [огонь](/tag/#огонь) ([луч](/tag/#луч)): оба его имеют, но один умеет им пользоваться, а другой – нет. Надо [учиться](/tag/#учение) владеть огнями. Овладеть, для того чтобы уметь применять в [жизни](/tag/#жизнь). Для [жизни](/tag/#жизнь) дается огненное могущество. Ясность Лика [Учителя](/tag/#учитель) ставит нас с Ним в непосредственное соприкосновение. Углубляйте ясность.
