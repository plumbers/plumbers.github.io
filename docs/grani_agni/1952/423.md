[Грани Агни Йоги 1952г](/grani_agni/1952)

Что такое микрокосм [человека](/tags/человек)? Микрокосм [человека](/tags/человек) есть орудие познавания Макрокосма. Познавание есть расширение и рост. Круг [сознания](/tags/сознание) увеличивается, пока не станет планетным, затем кругом системы солнечной и, наконец, [Космоса](/tags/космос), то есть Макрокосмом. Слияние микрокосма с Макрокосмом есть цель эволюции, то есть расширение [сознания](/tags/сознание) до [Беспредельности](/tags/планета). Идем через тысячелетия к Вечности в спирали вечного восхождения. И Ведущий – Я.
