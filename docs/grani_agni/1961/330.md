[Грани Агни Йоги 1961г](/grani_agni/1961)

Сила [Учителя](/tags/учитель) уявляется в [спокойствии](/tags/спокойствие). Каждое мгновение дает [возможность](/tags/возможности) применения [Учения](/tags/учение) в [жизни](/tags/жизнь). [Мысль](/tags/мысль) есть явление сущего счастья. Изменения внешних [условий](/tags/условие) [жизни](/tags/жизнь) нужны для сдвига [сознания](/tags/сознание). Хотящий имеет.
