[Грани Агни Йоги 1965г](/grani_agni/1965)

Моменты взлета [духа](/tags/Дух) будем ценить, чтобы использовать их [возможно](/tags/возможности) полнее. Не всегда [Свет](/tags/сознание) есть свой ритм, характеризующийся волнами Света, которые чередуются в определенной последовательности. Ритм этот ручательством служит того, что, как бы ни казалось темно, неизбежно наступление Света.
