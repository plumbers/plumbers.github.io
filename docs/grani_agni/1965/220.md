Каждое страдание приносит опыт. Результат [опыта](../../../tags/#опыт) выливается в записи. Следовательно, страдания есть плата за [возможность](../../../tags/#возможности) сделать новую запись и оставить ее на пользу тем, кто пойдет после. Не для себя, но других заполняются страницы тетради новыми записями, и за каждую заплачено полной ценой. Оценят их те, кто следовать будет тем же [путем](../../../tags/#путь). Только идти им будет много уже легче по дороге, проложенной раньше для них.
