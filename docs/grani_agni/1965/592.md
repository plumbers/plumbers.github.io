[Грани Агни Йоги 1965г](/grani_agni/1965)

Вера есть предвестница [знания](/tags/вещь) и планетным задачам эволюции. Сферы эволюционные есть сферы Света. Те, кто потрудились для эволюции, входят в сферы магнитного притяжения Света.
