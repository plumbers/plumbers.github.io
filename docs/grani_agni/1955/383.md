[Грани Агни Йоги 1955г](https://127.0.0.1:4002/grani_agni/1955)

___383___

Радуюсь, потому что [причин](../../../tags/#причина) для [радости](../../../tags/#[радость](../../../tags/#радость)) нет никаких. Радуюсь, потому что темно вокруг. Радуюсь, потому что нерадостно окружающее. Радуюсь, потому что хочу, ибо хочу утвердить [радость](../../../tags/#радость) вопреки вибрациям плотной среды, ибо ее волны хочу преодолеть волною своею, ибо хочу силу [духа](../../../tags/#Дух) поставить над плотью.
