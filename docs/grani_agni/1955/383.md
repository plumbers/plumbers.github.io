[Грани Агни Йоги 1955г](/grani_agni/1955)

___383___
Радуюсь, потому что [причин](/tag/#причина) для [радости](/tag/#[радость](/tag/#радость)) нет никаких. Радуюсь, потому что темно вокруг. Радуюсь, потому что нерадостно окружающее. Радуюсь, потому что хочу, ибо хочу утвердить [радость](/tag/#радость) вопреки вибрациям плотной среды, ибо ее волны хочу преодолеть волною своею, ибо хочу силу [духа](/tag/#Дух) поставить над плотью.
