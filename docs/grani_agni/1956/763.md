[Грани Агни Йоги 1956г](https://127.0.0.1:4002/grani_agni/1956)

Царство Мое есть Царство Света. Нельзя устремиться к нему, будучи тьмою объятым. Нужны одежды Света. В праздничные дни люди обычно надевают свои лучшие одежды. Тем более они нужны для праздника [духа](../../../tags/#Дух), ибо коснуться Меня есть праздник для [духа](../../../tags/#Дух). Позаботьтесь о новых одеждах, если хотите войти в Царство Мое.
