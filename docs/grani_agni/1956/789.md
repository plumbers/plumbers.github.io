[Грани Агни Йоги 1956г](/grani_agni/1956)

Пьянство есть враг психотехники. Пьянство разрушает провода нервов, делает их непригодными для тонких восприятий. Посредствующим звеном передачи огненных посылок служит нервная система. Если она не в порядке, высшие [возможности](/tags/возможности) утрачиваются. Вино разрушает [волю](/tags/сила).
