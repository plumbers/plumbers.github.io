[Грани Агни Йоги 1956г](/grani_agni/1956)

___70___
Сын Мой, много [условий](/tag/#условие) способствуют Общению, но главное из них – это [огни](/tag/#огонь) [сердца](/tag/#сердце), то есть зажженные центры. Центры не загораются сразу. Плоды многих [напряженных](/tag/#напряжение) [жизней](/tag/#жизнь) лежат в основе этого феномена. И никто не знает, когда [приходит](/tag/#приход) время проснуться для каждого данного центра. Но Знает [Учитель](/tag/#учитель), ибо Видит. Ступень [духа](/tag/#Дух) определяется по огням. [Огонь](/tag/#огонь) утвердившийся есть вечное и неотъемлемое приобретение. Восходим огнями. Без них нет продвижения. Потому на смене рас Дается Огненная Йога как сужденный [путь](/tag/#путь) эволюции [духа](/tag/#Дух).
