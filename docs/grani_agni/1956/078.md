[Грани Агни Йоги 1956г](/grani_agni/1956)

Возмущение вод необходимо, ибо в результате освобождаются дотоль связанные [энергии](/tags/энергия). Их и можно направлять по своему усмотрению. Момент этот надо суметь уловить, ибо этот момент наивыгоднейший для [действия](/tags/действие) можно разбрасывать в урочный час. Для этого требуется настороженность и чуткая зоркость.
