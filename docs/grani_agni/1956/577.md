[Грани Агни Йоги 1956г](/grani_agni/1956)

Заградительная сеть. [Огни](/tags/огонь) заградительной сети нагнетаются сознанием. Их можно либо совсем потушить, либо зажечь напряженно. Угашение огней заградительной сети означает полную беззащитность [человека](/tags/человек). [Волею](/tags/истина), воина в каждом видеть Хочу.
