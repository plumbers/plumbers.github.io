[Грани Агни Йоги 1956г](/grani_agni/1956)

___845___
Можно подумать и о том, как [волю](/tag/#[воля](/tag/#воля)) свою укрепить ритмом, ведь [равновесие](/tag/#равновесие) есть [воля](/tag/#воля), собранная на острие тарана и готовая для [действия](/tag/#действие). Это есть [огонь](/tag/#огонь), сосредоточенный для удара в нужном направлении, но обузданный [волей](/tag/#воля) и сдерживаемый ею для [возможности](/tag/#возможности) мощного [действия](/tag/#действие). [Воля](/tag/#воля) и [равновесие](/tag/#равновесие) связаны тесно. Не может являть [равновесия](/tag/#равновесие) безвольное существо. [Равновесие](/tag/#равновесие) есть [воля](/tag/#воля), собранная в фокус для [действия](/tag/#действие). Оба [качества](/tag/#качество) стоят того, чтобы [сил](/tag/#сила) не жалеть их утверждать.
