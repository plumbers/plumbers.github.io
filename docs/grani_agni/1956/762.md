[Грани Агни Йоги 1956г](/grani_agni/1956)

___762___
Есть [мысли](/tag/#[мысль](/tag/#мысль)) светлые, есть серые, есть темные. Каждая соединяет пространственно с резервуаром [мыслей](/tag/#[мысль](/tag/#мысль)) созвучных. Потому каждая [мысль](/tag/#мысль) есть провод, ведущий ко тьме или Свету. Когда очень темно, подумайте о Свете.
