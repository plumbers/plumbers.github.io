[Грани Агни Йоги 1956г](/grani_agni/1956)

Есть [мысли](/tags/[мысль](/tags/мысль)) светлые, есть серые, есть темные. Каждая соединяет пространственно с резервуаром [мыслей](/tags/[мысль](/tags/мысль)) созвучных. Потому каждая [мысль](/tags/мысль) есть провод, ведущий ко тьме или Свету. Когда очень темно, подумайте о Свете.
