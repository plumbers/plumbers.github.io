[Грани Агни Йоги 1956г](/grani_agni/1956)

___627___
[Луч](/tag/#луч) [действует](/tag/#действие). Надо только сознательно принять его (в [сознание](/tag/#сознание)). Пиши о новой ступени [сознания](/tag/#сознание), о [возможности](/tag/#возможности) сознательной ассимиляции Лучей [Матери](/tag/#материя) Мира. Сила сближения явно дает себя ощущать. Осознается иерархически, то есть по линии Иерархии (по Цепи Иерархии). Нет! [Дух](/tag/#Дух) твой свободен. Им действуй, духом. [Действию](/tag/#действие) [духа](/tag/#Дух) предпошли осознание его безграничных [возможностей](/tag/#возможности). Все сбудется. Храни ярое [Единение](/tag/#единение) с Нами.
