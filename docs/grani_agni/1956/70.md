[Грани Агни Йоги 1956г](https://127.0.0.1:4002/grani_agni/1956)

___70___

Сын Мой, много [условий](../../../tags/#условие) способствуют Общению, но главное из них – это [огни](../../../tags/#огонь) [сердца](../../../tags/#сердце), то есть зажженные центры. Центры не загораются сразу. Плоды многих [напряженных](../../../tags/#напряжение) [жизней](../../../tags/#жизнь) лежат в основе этого феномена. И никто не знает, когда [приходит](../../../tags/#приход) время проснуться для каждого данного центра. Но Знает [Учитель](../../../tags/#учитель), ибо Видит. Ступень [духа](../../../tags/#Дух) определяется по огням. [Огонь](../../../tags/#огонь) утвердившийся есть вечное и неотъемлемое приобретение. Восходим огнями. Без них нет продвижения. Потому на смене рас Дается Огненная Йога как сужденный [путь](../../../tags/#путь) эволюции [духа](../../../tags/#Дух).
