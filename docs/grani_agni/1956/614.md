[Грани Агни Йоги 1956г](/grani_agni/1956)

Жизнь есть утверждение того, чем хочет стать [дух](/tags/Дух). Это для [духа](/tags/Дух) сознательного. Для него преображение себя в процессе [жизни](/tags/жизнь) и будет ее целью. Много может быть целей в [жизни](/tags/жизнь), но это – главное, что делаете вы.
