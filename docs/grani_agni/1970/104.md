[Грани Агни Йоги 1970г](/grani_agni/1970)

___104___
При всей глубине сочувствия чужому страданию нельзя позволять себе утрачивать [равновесие](/tag/#равновесие), ибо утрата его делает [человека](/tag/#человек) беспомощным, то есть неспособным оказать реальную [помощь](/tag/#помощь). Потому сочувствие допускается лишь при условии незаражения омраченным состоянием страдающего. Спокойная, светлая, ясная [аура](/tag/#аура) оказывает действительную [помощь](/tag/#помощь), если светоносность не нарушается эмоциональным неуравновесием страдальца.
