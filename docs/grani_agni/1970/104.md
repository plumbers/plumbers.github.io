[Грани Агни Йоги 1970г](https://127.0.0.1:4002/grani_agni/1970)

___104___

При всей глубине сочувствия чужому страданию нельзя позволять себе утрачивать [равновесие](../../../tags/#равновесие), ибо утрата его делает [человека](../../../tags/#человек) беспомощным, то есть неспособным оказать реальную [помощь](../../../tags/#помощь). Потому сочувствие допускается лишь при условии незаражения омраченным состоянием страдающего. Спокойная, светлая, ясная [аура](../../../tags/#аура) оказывает действительную [помощь](../../../tags/#помощь), если светоносность не нарушается эмоциональным неуравновесием страдальца.
