[Беспредельность 1 1930г](/agni/1930)

___288___
Токи [Космического](/tag/#космос) Магнита напрягают все течения явленные во Вселенной. Комбинации творческих [сил](/tag/#сила) утверждаются на всех сферах. Творчество [Космического](/tag/#космос) Магнита утверждает различие элементов, нужных для сочетания психозёрен. Психозёрна собирают те элементы вокруг себя, которые нужны для формирования течения предопределённого. Так психозёрна могут сформировать ту сферу, которая заложена в ядре. Токи тончайших [энергий](/tag/#энергия) собираются вокруг ядра, которое содержит беспредельные начинания.   

[Сердце](/tag/#[сердце](/tag/#сердце)) Тары устремляет явление подвига. [Сердце](/tag/#[сердце](/tag/#сердце)) Тары устремляет лучами к Свету. [Сердце](/tag/#[сердце](/tag/#сердце)) Тары даёт токи Нашим устремлениям. Утверждаю [сердце](/tag/#сердце) Нашей [Урусвати](/tag/#Урусвати) как [свет](/tag/#свет) Наших [устремлений](/tag/#устремление). Да, да, да!
