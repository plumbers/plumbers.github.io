[Беспредельность 1 1930г](/agni/1930)

Токи [Космического](/tags/космос) Магнита напрягают все течения явленные во Вселенной. Комбинации творческих [сил](/tags/сила) утверждаются на всех сферах. Творчество [Космического](/tags/космос) Магнита утверждает различие элементов, нужных для сочетания психозёрен. Психозёрна собирают те элементы вокруг себя, которые нужны для формирования течения предопределённого. Так психозёрна могут сформировать ту сферу, которая заложена в ядре. Токи тончайших [энергий](/tags/энергия) собираются вокруг ядра, которое содержит беспредельные начинания.
