[Беспредельность 1 1930г](/agni/1930)

Когда сопротивление [Космическому](/tags/космос) Магниту происходит для собирания новых сочетаний для временного явления, это явление совершается. [Космический](/tags/космос) Магнит сцепляет те сочетания, которые уносятся в пространственное колесо творчества. Сила [Космического](/tags/космос) Магнита центробежна и выбрасывает те части, не соответствующие главной задаче — соединению тех частей, которые [образуют](/tags/образ) из свойственных одному элементу [качеств](/tags/качество) одно целое. Сопротивление [духа](/tags/Дух) Началам Вселенной так жестоко погрузило сосуд [знания](/tags/космос) Магнит собирает свой Высший Разум беспредельного творчества!
