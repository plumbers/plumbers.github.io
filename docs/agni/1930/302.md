[Беспредельность 1 1930г](https://127.0.0.1:4002/agni/1930)

___302___
Все проявления Совершенного [Сердца](../../../tags/#сердце) соответствуют всем огням, которые [дух](../../../tags/#Дух) устремляет на Общее [Благо](../../../tags/#благо). [Огни](../../../tags/#огонь) [духа](../../../tags/#Дух) напрягаются центрами высших сфер; потому общение [духа](../../../tags/#Дух) с сферами утверждается.   

Магнит Совершенного [Сердца](../../../tags/#сердце) предусматривает общение с духом устремлённым. Куда же направится светоносный [луч](../../../tags/#луч) Совершенного [Сердца](../../../tags/#сердце)? Каждый сознательный ответ есть [вибрация](../../../tags/#вибрация) тождественности. Потому эманации центров в свою очередь направляют, вызывая сознательную устремлённость. Центры творческого [огня](../../../tags/#огонь) строят ступени [человечества](../../../tags/#человечество). Потому так разнообразно устремляются [огни](../../../tags/#огонь) [Агни-йога](../../../tags/#Агни-Йога).
