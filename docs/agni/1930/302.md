[Беспредельность 1 1930г](/agni/1930)

___302___
Все проявления Совершенного [Сердца](/tag/#сердце) соответствуют всем огням, которые [дух](/tag/#Дух) устремляет на Общее [Благо](/tag/#благо). [Огни](/tag/#огонь) [духа](/tag/#Дух) напрягаются центрами высших сфер; потому общение [духа](/tag/#Дух) с сферами утверждается.   

Магнит Совершенного [Сердца](/tag/#сердце) предусматривает общение с духом устремлённым. Куда же направится светоносный [луч](/tag/#луч) Совершенного [Сердца](/tag/#сердце)? Каждый сознательный ответ есть [вибрация](/tag/#вибрация) тождественности. Потому эманации центров в свою очередь направляют, вызывая сознательную устремлённость. Центры творческого [огня](/tag/#огонь) строят ступени [человечества](/tag/#человечество). Потому так разнообразно устремляются [огни](/tag/#огонь) [Агни-йога](/tag/#Агни-Йога).
