[Беспредельность 1 1930г](/agni/1930)

Творчество [Беспредельности](/tags/беспредельность) на дальних мирах явит самые разнообразные сочетания. У каждого элемента, скомбинированного с элементом родственным, есть все данные для лучших комбинаций. Вместо [работы](/tags/работа) взрывами лабораторными, лучи будут всё трансформировать.   

[Огни](/tags/значение) явления огней беспредельно.
