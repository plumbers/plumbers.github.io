[Беспредельность 1 1930г](/agni/1930)

___269___
В беспредельном созвучии заключено всё [космическое](/tag/#космос) творчество. Только созвучие может сферы высшие явить планете. Только созвучие может утвердить цепь [устремлений](/tag/#устремление) к дальним мирам. [Понятие](/tag/#понятие) дальних миров нужно применить ко всему [устремлению](/tag/#устремление) к совершенствованию. Потому Наши созвучия так мощны. [Огонь](/tag/#огонь) центров есть созвучие. [Огонь](/tag/#огонь) [духа](/tag/#Дух) есть созвучие. Пламя [сердца](/tag/#сердце) есть созвучие.
