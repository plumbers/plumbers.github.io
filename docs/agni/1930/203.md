[Беспредельность 1 1930г](/agni/1930)

___203___
Жемчужина [сердца](/tag/#сердце) — тончайшая напряжённость. Только этой напряжённостью создаём миры. Ведь [мир](/tag/#мир) [устремлений](/tag/#устремление) строит [будущее](/tag/#будущее). Источник любви объединяет сущее!   

Скажу, [истинно](/tag/#истина), облик [Матери](/tag/#материя) Агни-Йоги облечён в Материю Люциду. И Моя [Рука](/tag/#[рука](/tag/#рука)) свидетельствует, как [рука](/tag/#рука) Тары творит.   

Гуру должен почитаться, и доверие явленное должно хранить как ручательство Начал. Сотрудникам нужно явить справедливость к себе — суровое отношение к себе.   

Глаз добрый творит. Страж может продвигаться, только судя себя, не судя словами Гуру. Гуру направляет [добрым](/tag/#добро) глазом. Мало понять [слова](/tag/#слово), затмив их властностью. Гуру пусть чтут — не для своих выгод, но для общего [блага](/tag/#благо).
