[Беспредельность 1 1930г](https://127.0.0.1:4002/agni/1930)

Сферы подземные соответствуют порождениям [человечества](../../../tags/#человечество). Каждое [действие](../../../tags/#действие), каждая [мысль](../../../tags/#мысль) имеет свои зёрна. К этим зёрнам притягиваются все соответственные элементы. Потому все [человеческие](../../../tags/#человечество) порождения так отвечают [земным](../../../tags/#Земля) мыслям. Потому низшие сферы так напоминают тёмные [устремления](../../../tags/#[устремление](../../../tags/#устремление)). Видела [Урусвати](../../../tags/#Урусвати) порождения [земные](../../../tags/#Земля), видела затверделых гигантов, которые символизируют [устремление](../../../tags/#устремление) [человечества](../../../tags/#человечество). Сфера [человеческих](../../../tags/#человечество) порождений разделяется пластами, и посещение таких сфер сопряжено с бронированием огненным. Сферы подземные есть выражение [человеческих](../../../tags/#человечество) [мыслей](../../../tags/#мысль). Мощный [огонь](../../../tags/#огонь) [духа](../../../tags/#Дух) есть разбивающий молот. [Огонь](../../../tags/#огонь) [устремления](../../../tags/#[устремление](../../../tags/#устремление)) [духа](../../../tags/#Дух) может Вселенную преобразить. Потому символ молота есть развитие новых [напряжений](../../../tags/#напряжение). [Земные](../../../tags/#Земля) подпруги рушатся, и основание новое закладывается. Сила [огня](../../../tags/#огонь) [Матери](../../../tags/#материя) Агни-Йоги, [истинно](../../../tags/#истина), может создать [устремление](../../../tags/#устремление) [человечества](../../../tags/#человечество)!
