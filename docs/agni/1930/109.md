[Беспредельность 1 1930г](/agni/1930)

___109___
Преображение является Нашим прекраснейшим основанием [космического](/tag/#космос) слияния. Ту ступень [космического](/tag/#космос) преображения, когда человек завершает [земной](/tag/#[Земля](/tag/#Земля)) [путь](/tag/#путь), Мы называем [космической](/tag/#космос) радостью. Явление завершения — самое трудное достижение. Трансмутация на высших сферах гораздо легче. Самый тяжёлый разрыв с Землёю, [Земля](/tag/#Земля) будучи утверждением самым высоким из цепи низших миров. Самый подъём тяжек на высшие сферы. Потому так медленно и редко [дух](/tag/#Дух) завершает [путь](/tag/#путь).   

[Планеты](/tag/#планета) высшие не имеют того тяготения. [Устремление](/tag/#устремление) есть двигатель; жизненное пламенное объединение уносит в сферы высшие. [Путь](/tag/#путь) [земной](/tag/#[Земля](/tag/#Земля)) тяжек духу завершающему.
