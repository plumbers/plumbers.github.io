[Беспредельность 1 1930г](/agni/1930)

___270___
Соответствие между устремлением и следствием определённо утверждает форму, которая является высшей напряжённостью. Сродство [устремления](/tag/#устремление) с притяжением может проявить формулу, отвечающую силе [вибрации](/tag/#вибрация) [огня](/tag/#огонь) [пространства](/tag/#[пространство](/tag/#пространство)). Вмещение и проявление соотносятся. Потому сила, употреблённая на проявление [энергии](/tag/#энергия), даёт явление соответствия.   

Если бы [человечество](/tag/#человечество) поняло, что только творчество насыщает [пространство](/tag/#пространство), то оно применило бы [устремление](/tag/#устремление), умноженное сознательным огнём. Только ту величину можно извлекать из творчества [Космического](/tag/#космос) Магнита. Только токи напряжённые, как струны [космические](/tag/#космос), дадут формулу уявленного Магнита [Беспредельности](/tag/#беспредельность)!
