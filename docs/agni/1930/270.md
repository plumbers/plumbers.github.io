[Беспредельность 1 1930г](/agni/1930)

Соответствие между устремлением и следствием определённо утверждает форму, которая является высшей напряжённостью. Сродство [устремления](/tags/устремление) с притяжением может проявить формулу, отвечающую силе [вибрации](/tags/вибрация) [огня](/tags/огонь) [пространства](/tags/энергия), даёт явление соответствия.   

Если бы [человечество](/tags/человечество) поняло, что только творчество насыщает [пространство](/tags/пространство), то оно применило бы [устремление](/tags/устремление), умноженное сознательным огнём. Только ту величину можно извлекать из творчества [Космического](/tags/космос) Магнита. Только токи напряжённые, как струны [космические](/tags/космос), дадут формулу уявленного Магнита [Беспредельности](/tags/беспредельность)!
