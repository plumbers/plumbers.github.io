# 235

Потому каждый [дух](/tags/Дух) должен понять Иерархию как Покровителей [человечества](/tags/человечество); так слагается [эволюция](/tags/эволюция) и творчество насыщается огненно. Так утверждается [закон](/tags/закон); только так насыщается жизнь великою мощью объединения. Так строится жизнь.
