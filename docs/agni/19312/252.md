[Иерархия 1931г](/agni/19312)

# 252

Потому когда [устремление](/tags/устремление) насыщает [дух](/tags/Дух), то соответственно устанавливается [понимание](/tags/понимание) необходимости высших [мер](/tags/мера). Потому для продвижения нужно понять, как бережно относиться к Иерархии; потому нужно утвердить [сознание](/tags/сознание) на великом [понимании](/tags/понимание) Иерархии. Так без явленного фокуса не достичь и не построить. Творчество может развиться только утверждением фокуса, так заповедаю!
