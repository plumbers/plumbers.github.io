[Мир огненный 1 1933г](/agni/1933)

___251___
Каждый физический [аппарат](/tag/#аппарат) даёт совершенную аналогию в Мире Тонком. Притом можно легко чувствовать, как нетрудно увеличить [силы](/tag/#сила) аппарата, призвав Агни. Так можно заново восстановить множество [опытов](/tag/#опыт), казавшихся неудачными. [Опыты](/tag/#опыт) Кили и даже [аппарат](/tag/#аппарат) Эдисона для Тонкого Мира остались несовершенными, ибо [энергия](/tag/#энергия) Агни не была приложена; в одном случае по окружающей подозрительности, в другом — по личному недоверию. Сказано: даже свеча не зажжётся без доверия.
