[Мир огненный 1 1933г](/agni/1933)

Древние союзы скреплялись прыганием через [огонь](/tags/огонь). Для клятвы держали [руку](/tags/рука) над огнём. Для освящения проходили через [огонь](/tags/огонь). Подобное свидетельство огнём прошло по всем векам. Нужно принять это как основание огненной стихии очищения. Нужно и в [мыслях](/tags/качество) уже будет нужно для самого Огненного Мира.
