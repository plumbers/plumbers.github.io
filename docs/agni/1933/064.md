[Мир огненный 1 1933г](/agni/1933)

Нужно трудные распознания токов понять. Многие не умели бы различить сложное изменение токов и ритмов. Очень хвалю [Урусвати](/tags/Урусвати) за [внимание](/tags/внимание) к токам, — так только можно накоплять наблюдения. Через два года можно будет сообщить один из самых сложных токов, который без предварительного накопления невыносим.
