[Мир огненный 1 1933г](/agni/1933)

Свобода выбора заложена во всём. Никакое насилие не должно прерывать [путь](/tags/путь), но позволительно дать каждому светильник в [пути](/tags/путь) долгом. Просвещение лишь может осмыслить [свободу](/tags/свобода) избрания, потому просвещение есть утверждение бытия. Каждая школа должна с малых лет давать просвещение, связывая действительность с сущностью суждённого. Лишь так можем связать наше бытие с усовершенствованием. Свобода выбора, просвещение, самоусовершенствование есть [путь](/tags/путь) [Огня](/tags/огонь). Только огненные существа могут самостоятельно почувствовать эти устои восхождения. Но всех нужно вести этими вратами, иначе откуда же разрушительные смятения, которые вместе с хаосом стихий заставляют трепетать [планету](/tags/планета)? Так разнузданные [человеческие](/tags/человечество) смятения присоединяются к стихийному возмущению. Считаю, нужно твердить о смятении, которое сокрушает все зачатки эволюции.
