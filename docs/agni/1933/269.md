[Мир огненный 1 1933г](/agni/1933)

___269___
Свобода выбора заложена во всём. Никакое насилие не должно прерывать [путь](/tag/#путь), но позволительно дать каждому светильник в [пути](/tag/#путь) долгом. Просвещение лишь может осмыслить [свободу](/tag/#свобода) избрания, потому просвещение есть утверждение бытия. Каждая школа должна с малых лет давать просвещение, связывая действительность с сущностью суждённого. Лишь так можем связать наше бытие с усовершенствованием. Свобода выбора, просвещение, самоусовершенствование есть [путь](/tag/#путь) [Огня](/tag/#огонь). Только огненные существа могут самостоятельно почувствовать эти устои восхождения. Но всех нужно вести этими вратами, иначе откуда же разрушительные смятения, которые вместе с хаосом стихий заставляют трепетать [планету](/tag/#планета)? Так разнузданные [человеческие](/tag/#человечество) смятения присоединяются к стихийному возмущению. Считаю, нужно твердить о смятении, которое сокрушает все зачатки эволюции.
