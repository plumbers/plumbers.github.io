Спросят: если для Хатха-Йоги нужны упражнения [тела](../../../tags/#[тело](../../../tags/#тело)), то для других Йог нужны ли эти движения? Не имели их Архаты и подвижники. Действительно, они имеют испытания [духа](../../../tags/#Дух), которые не только подчиняют [тело](../../../tags/#тело), но заменяют для него упражнение плоти. Только признание [духа](../../../tags/#Дух) может заменить остальное.
