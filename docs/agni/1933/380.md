[Мир огненный 1 1933г](/agni/1933)

___380___
Спросят: если для Хатха-Йоги нужны упражнения [тела](/tag/#[тело](/tag/#тело)), то для других Йог нужны ли эти движения? Не имели их Архаты и подвижники. Действительно, они имеют испытания [духа](/tag/#Дух), которые не только подчиняют [тело](/tag/#тело), но заменяют для него упражнение плоти. Только признание [духа](/tag/#Дух) может заменить остальное.
