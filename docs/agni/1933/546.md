[Мир огненный 1 1933г](/agni/1933)

Отдавание есть божественный признак. Неистощимость отдавания проходит в разных степенях во всей природе. Но [огонь](/tags/огонь) будет самою яркою в отдаче стихией. Сам принцип [Огня](/tags/огонь) есть претворение и постоянное отдаяние. [Огонь](/tags/огонь) не может существовать без [жертвы](/tags/мир) Огненный не может быть помыслен иначе как явление величественности.
