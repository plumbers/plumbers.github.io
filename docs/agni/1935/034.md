[Мир огненный 3 1935г](/agni/1935)

___34___
Среди огненных борений [дух](/tag/#Дух) являет тоску. Особенно знает [дух](/tag/#Дух) эти борения на последней ступени. Тоска есть явление Тонкого Мира, и неудовлетворенность есть [знание](/tag/#знание) [будущего](/tag/#будущее). При делимости [духа](/tag/#Дух) это [чувство](/tag/#чувство) особенно мощно.
