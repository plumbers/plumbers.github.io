[Мир огненный 3 1935г](/agni/1935)

Среди огненных борений [дух](/tags/Дух) являет тоску. Особенно знает [дух](/tags/Дух) эти борения на последней ступени. Тоска есть явление Тонкого Мира, и неудовлетворенность есть [знание](/tags/знание) [будущего](/tags/будущее). При делимости [духа](/tags/Дух) это [чувство](/tags/чувство) особенно мощно.
