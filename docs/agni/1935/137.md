[Мир огненный 3 1935г](/agni/1935)

Живая Этика заключает в себе законы для явлений [Истины](/tags/истина). Жизнь утверждается на всех высших понятиях; так творчество Живой Этики направляет [мысль](/tags/мысль) к созиданию сущего. Все [устремления](/tags/энергия) во имя Живой Этики.
