[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___154___

[Качество](../../../tags/#качество) [духа](../../../tags/#Дух) — распознание — может рассматриваться по тем действиям, которые особенно ярко раскрывают недра [сердца](../../../tags/#сердце). Именно там, где отсутствует смирение, там будет [место](../../../tags/#место) самозванству. Там, где Иерархия не почитается, там будет [место](../../../tags/#место) кощунству. Там, где Указ Высших [Сил](../../../tags/#сила) утверждается лишь своевольно, там кроется самость. Но где Огненный [Учитель](../../../tags/#учитель) отсутствует, направление не будет в сторону [Учения](../../../tags/#учение). Невозможно осознать великого [Учения](../../../tags/#учение) без Огненного [Учителя](../../../tags/#учитель), без [устремления](../../../tags/#устремление) [духа](../../../tags/#Дух) к Миру [Учителя](../../../tags/#учитель). Явление Огненного [Учителя](../../../tags/#учитель) есть [путь](../../../tags/#путь) к Миру Огненному. Так рекорды [пространства](../../../tags/#пространство) наполняются самовольными Учителями, но Ведущее Начало есть Огненный [Учитель](../../../tags/#учитель). Не пройти без Него, не продвинуться без Него, не достичь без Него. Так запомним, творя лучшее [будущее](../../../tags/#будущее).
