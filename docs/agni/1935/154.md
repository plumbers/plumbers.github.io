[Мир огненный 3 1935г](/agni/1935)

___154___
[Качество](/tag/#качество) [духа](/tag/#Дух) — распознание — может рассматриваться по тем действиям, которые особенно ярко раскрывают недра [сердца](/tag/#сердце). Именно там, где отсутствует смирение, там будет [место](/tag/#место) самозванству. Там, где Иерархия не почитается, там будет [место](/tag/#место) кощунству. Там, где Указ Высших [Сил](/tag/#сила) утверждается лишь своевольно, там кроется самость. Но где Огненный [Учитель](/tag/#учитель) отсутствует, направление не будет в сторону [Учения](/tag/#учение). Невозможно осознать великого [Учения](/tag/#учение) без Огненного [Учителя](/tag/#учитель), без [устремления](/tag/#устремление) [духа](/tag/#Дух) к Миру [Учителя](/tag/#учитель). Явление Огненного [Учителя](/tag/#учитель) есть [путь](/tag/#путь) к Миру Огненному. Так рекорды [пространства](/tag/#пространство) наполняются самовольными Учителями, но Ведущее Начало есть Огненный [Учитель](/tag/#учитель). Не пройти без Него, не продвинуться без Него, не достичь без Него. Так запомним, творя лучшее [будущее](/tag/#будущее).
