[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___254___
[Воля](../../../tags/#воля) воинствующего [духа](../../../tags/#Дух) может направить целое воинство ко благу. [Воля](../../../tags/#воля) воинствующего [духа](../../../tags/#Дух) может направить целый воинствующий [мир](../../../tags/#мир). [Воля](../../../tags/#воля) воинствующего [духа](../../../tags/#Дух) может утвердить новые каналы для проведения строительства, потому каждая стена может разрушиться под напором воинствующей [воли](../../../tags/#воля). Воинствующий [дух](../../../tags/#Дух), открывающий огненный горизонт, есть [дух](../../../tags/#Дух) утверждающий Мощь Высшую. Воинствующий [дух](../../../tags/#Дух) может насытить каждое явление, идущее с [Космическим](../../../tags/#космос) Магнитом. Сколько [напряжений](../../../tags/#напряжение) может [дух](../../../tags/#Дух) воинствующий превозмочь! [Воля](../../../tags/#воля) воинствующая создает и творит новые [возможности](../../../tags/#возможности). Так пусть все, идущие к Свету, поймут [значение](../../../tags/#значение) направляющей [Воли](../../../tags/#воля) воинствующей, ибо идущие за [Волей](../../../tags/#воля) воинствующей следуют за [Огнем](../../../tags/#огонь). Так явим [понимание](../../../tags/#понимание) огненным носителям Нашей [Воли](../../../tags/#воля).
