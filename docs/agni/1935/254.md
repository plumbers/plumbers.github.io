[Мир огненный 3 1935г](/agni/1935)

___254___
[Воля](/tag/#воля) воинствующего [духа](/tag/#Дух) может направить целое воинство ко благу. [Воля](/tag/#воля) воинствующего [духа](/tag/#Дух) может направить целый воинствующий [мир](/tag/#мир). [Воля](/tag/#воля) воинствующего [духа](/tag/#Дух) может утвердить новые каналы для проведения строительства, потому каждая стена может разрушиться под напором воинствующей [воли](/tag/#воля). Воинствующий [дух](/tag/#Дух), открывающий огненный горизонт, есть [дух](/tag/#Дух) утверждающий Мощь Высшую. Воинствующий [дух](/tag/#Дух) может насытить каждое явление, идущее с [Космическим](/tag/#космос) Магнитом. Сколько [напряжений](/tag/#напряжение) может [дух](/tag/#Дух) воинствующий превозмочь! [Воля](/tag/#воля) воинствующая создает и творит новые [возможности](/tag/#возможности). Так пусть все, идущие к Свету, поймут [значение](/tag/#значение) направляющей [Воли](/tag/#воля) воинствующей, ибо идущие за [Волей](/tag/#воля) воинствующей следуют за [Огнем](/tag/#огонь). Так явим [понимание](/tag/#понимание) огненным носителям Нашей [Воли](/tag/#воля).
