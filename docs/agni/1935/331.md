[Мир огненный 3 1935г](/agni/1935)

___331___
Те [вибрации](/tag/#[вибрация](/tag/#вибрация)), которые насыщают [Мир](/tag/#мир), отвечают свойствам низших проявлений. Усовершенствование планетных [вибраций](/tag/#[вибрация](/tag/#вибрация)) может происходить лишь напряжением [человечества](/tag/#[человечество](/tag/#человечество)). Центр всех выявлений есть [человечество](/tag/#человечество), которое отражает все слои. [Вибрации](/tag/#[вибрация](/tag/#вибрация)) настолько дисгармоничны, что трудно установить связь с Мирами Высшими. [Вибрации](/tag/#[вибрация](/tag/#вибрация)), способствующие разъединению, имеют [качество](/tag/#качество) самое низменное. Потому слои низшие могут пропускать свои низменные [вибрации](/tag/#[вибрация](/tag/#вибрация)), тогда как высшие излучения не доходят до [Земли](/tag/#Земля). Насыщенный [Мир](/tag/#мир) ждет великой эпохи Перерождения [Духа](/tag/#Дух). Насыщение [пространства](/tag/#пространство) вибрациями, помогающими установлению высших излучений, есть задача на [пути](/tag/#путь) к Миру Огненному.
