[Мир огненный 3 1935г](/agni/1935)

[Сущность](/tags/сущность) огненного продвижения состоит из разных утверждений [духа](/tags/Дух). Главный фактор есть развитие самодеятельности. В самодеятельности будет заключаться [любовь](/tags/любовь) к Иерархии, в ней будет заключаться [чувство](/tags/чувство) ответственности и [истинное](/tags/истина) [понимание](/tags/понимание) [Служения](/tags/служение) Иерархии есть высшее достижение.
