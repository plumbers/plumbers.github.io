[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___203___

Мосты [духа](../../../tags/#[Дух](../../../tags/#Дух)), переброшенные через все трудности, через все пропасти, идущими [путем](../../../tags/#путь) [устремления](../../../tags/#устремление), будут теми строительными энергиями в [Космосе](../../../tags/#космос). [Дух](../../../tags/#Дух), [истинно](../../../tags/#истина), может объединить разные центры. [Человечество](../../../tags/#[человечество](../../../tags/#человечество)) напрягается в видимом строительстве, которое не вдохновлено высшими устремлениями для [космического](../../../tags/#космос) строительства. Каждый мост [духа](../../../tags/#[Дух](../../../tags/#Дух)) утверждает, именно, сознательное строительство, которое являет [космическую](../../../tags/#космос) связь между построениями. Так в Эпоху Огненную нужно особенно отдаться тонкой, сознательной деятельности [духа](../../../tags/#[Дух](../../../tags/#Дух)). Лишь мост [духа](../../../tags/#[Дух](../../../tags/#Дух)) может уничтожить ту пропасть, которая разверзлась перед человечеством. Мост [Духа](../../../tags/#Дух) есть мост Красоты — этими понятиями подвига [духа](../../../tags/#[Дух](../../../tags/#Дух)) мы взойдем на великую Вершину, которая объединяет Миры.
