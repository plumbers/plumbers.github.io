[Мир огненный 3 1935г](/agni/1935)

___203___
Мосты [духа](/tag/#[Дух](/tag/#Дух)), переброшенные через все трудности, через все пропасти, идущими [путем](/tag/#путь) [устремления](/tag/#устремление), будут теми строительными энергиями в [Космосе](/tag/#космос). [Дух](/tag/#Дух), [истинно](/tag/#истина), может объединить разные центры. [Человечество](/tag/#[человечество](/tag/#человечество)) напрягается в видимом строительстве, которое не вдохновлено высшими устремлениями для [космического](/tag/#космос) строительства. Каждый мост [духа](/tag/#[Дух](/tag/#Дух)) утверждает, именно, сознательное строительство, которое являет [космическую](/tag/#космос) связь между построениями. Так в Эпоху Огненную нужно особенно отдаться тонкой, сознательной деятельности [духа](/tag/#[Дух](/tag/#Дух)). Лишь мост [духа](/tag/#[Дух](/tag/#Дух)) может уничтожить ту пропасть, которая разверзлась перед человечеством. Мост [Духа](/tag/#Дух) есть мост Красоты — этими понятиями подвига [духа](/tag/#[Дух](/tag/#Дух)) мы взойдем на великую Вершину, которая объединяет Миры.
