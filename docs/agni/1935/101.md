[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___101___
[Истинно](../../../tags/#истина), лишь [сердце](../../../tags/#сердце) может вникать во все [действия](../../../tags/#действие), во все побуждения, во все [сущности](../../../tags/#сущность), являя распознавание. Для проникновения в [Мир](../../../tags/#мир) Огненный нужно особенно распознавать сердцем. Лишь тот источник, который устремлен к основам [Истины](../../../tags/#истина), может дать [понятие](../../../tags/#понятие) истинного строения [Космоса](../../../tags/#космос). Только тот источник дает правильную [меру](../../../tags/#мер) суждений, который насыщается [огнем](../../../tags/#огонь) тонких [энергий](../../../tags/#[энергия](../../../tags/#энергия)). Для утверждения своих [сил](../../../tags/#сила) на Высших Сферах необходимо напрячь [силы](../../../tags/#сила) [сердца](../../../tags/#сердце), ибо нет другого свойства [Огня](../../../tags/#огонь), которое могло бы заменить эти [энергии](../../../tags/#[энергия](../../../tags/#энергия)). [Сердце](../../../tags/#сердце) так мощно устремляет [дух](../../../tags/#Дух) к тонким энергиям. Все Высшие Сферы достигаются напряжением [сердца](../../../tags/#сердце). Этот сокровенный сосуд может открыть все творческие возвышенные сферы. Незаменимы эти сердечные [энергии](../../../tags/#[энергия](../../../tags/#энергия)), в них Высшая [Воля](../../../tags/#воля), [истинно](../../../tags/#истина), отражается. Творчество [сердца](../../../tags/#сердце) можно назвать солнцеподобным. На [пути](../../../tags/#путь) к Миру Огненному устремимся к [пониманию](../../../tags/#понимание) [сердца](../../../tags/#сердце), как связующего явления между Мирами.
