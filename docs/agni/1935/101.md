[Мир огненный 3 1935г](/agni/1935)

[Истинно](/tags/истина), лишь [сердце](/tags/сердце) может вникать во все [действия](/tags/действие), во все побуждения, во все [сущности](/tags/сущность), являя распознавание. Для проникновения в [Мир](/tags/мир) Огненный нужно особенно распознавать сердцем. Лишь тот источник, который устремлен к основам [Истины](/tags/истина), может дать [понятие](/tags/понятие) истинного строения [Космоса](/tags/космос). Только тот источник дает правильную [меру](/tags/мер) суждений, который насыщается [огнем](/tags/огонь) тонких [энергий](/tags/сердце), как связующего явления между Мирами.
