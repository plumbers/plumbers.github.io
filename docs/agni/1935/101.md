[Мир огненный 3 1935г](/agni/1935)

___101___
[Истинно](/tag/#истина), лишь [сердце](/tag/#сердце) может вникать во все [действия](/tag/#действие), во все побуждения, во все [сущности](/tag/#сущность), являя распознавание. Для проникновения в [Мир](/tag/#мир) Огненный нужно особенно распознавать сердцем. Лишь тот источник, который устремлен к основам [Истины](/tag/#истина), может дать [понятие](/tag/#понятие) истинного строения [Космоса](/tag/#космос). Только тот источник дает правильную [меру](/tag/#мер) суждений, который насыщается [огнем](/tag/#огонь) тонких [энергий](/tag/#[энергия](/tag/#энергия)). Для утверждения своих [сил](/tag/#сила) на Высших Сферах необходимо напрячь [силы](/tag/#сила) [сердца](/tag/#сердце), ибо нет другого свойства [Огня](/tag/#огонь), которое могло бы заменить эти [энергии](/tag/#[энергия](/tag/#энергия)). [Сердце](/tag/#сердце) так мощно устремляет [дух](/tag/#Дух) к тонким энергиям. Все Высшие Сферы достигаются напряжением [сердца](/tag/#сердце). Этот сокровенный сосуд может открыть все творческие возвышенные сферы. Незаменимы эти сердечные [энергии](/tag/#[энергия](/tag/#энергия)), в них Высшая [Воля](/tag/#воля), [истинно](/tag/#истина), отражается. Творчество [сердца](/tag/#сердце) можно назвать солнцеподобным. На [пути](/tag/#путь) к Миру Огненному устремимся к [пониманию](/tag/#понимание) [сердца](/tag/#сердце), как связующего явления между Мирами.
