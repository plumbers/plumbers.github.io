[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___72___

Именно, [дух](../../../tags/#Дух) [человеческий](../../../tags/#человечество) есть проводник всех высших [энергий](../../../tags/#энергия). Как мощный провод, [дух](../../../tags/#Дух) являет разные функции для утверждения явлений Высшей [Воли](../../../tags/#воля). Как же иначе связать все проявления на разных планах? Лишь утверждения [духа](../../../tags/#Дух) могут являться звеном. Потому [сердце](../../../tags/#сердце) и [рука](../../../tags/#рука) [Учителя](../../../tags/#учитель) водящего усматривают все факторы, которые необходимы для восхождения. Среди огненных [понятий](../../../tags/#понятие) нужно особенно чуять величие звена, которое может связать миры. Из [рук](../../../tags/#рука) [Учителя](../../../tags/#учитель) водящего ученик получает напутствие к приобщению к мощи [Огня](../../../tags/#огонь). От [сердца](../../../tags/#сердце) [Учителя](../../../tags/#учитель) водящего общинник получает огненное [сознание](../../../tags/#сознание). Только [дух](../../../tags/#Дух) [человека](../../../tags/#человек) может, [истинно](../../../tags/#истина), связать миры. Так на [пути](../../../tags/#путь) к Миру Огненному явим огненное почитание земному [Учителю](../../../tags/#учитель), который закладывает зерно всего Высшего.
