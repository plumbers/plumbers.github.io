[Мир огненный 3 1935г](/agni/1935)

___72___
Именно, [дух](/tag/#Дух) [человеческий](/tag/#человечество) есть проводник всех высших [энергий](/tag/#энергия). Как мощный провод, [дух](/tag/#Дух) являет разные функции для утверждения явлений Высшей [Воли](/tag/#воля). Как же иначе связать все проявления на разных планах? Лишь утверждения [духа](/tag/#Дух) могут являться звеном. Потому [сердце](/tag/#сердце) и [рука](/tag/#рука) [Учителя](/tag/#учитель) водящего усматривают все факторы, которые необходимы для восхождения. Среди огненных [понятий](/tag/#понятие) нужно особенно чуять величие звена, которое может связать миры. Из [рук](/tag/#рука) [Учителя](/tag/#учитель) водящего ученик получает напутствие к приобщению к мощи [Огня](/tag/#огонь). От [сердца](/tag/#сердце) [Учителя](/tag/#учитель) водящего общинник получает огненное [сознание](/tag/#сознание). Только [дух](/tag/#Дух) [человека](/tag/#человек) может, [истинно](/tag/#истина), связать миры. Так на [пути](/tag/#путь) к Миру Огненному явим огненное почитание земному [Учителю](/tag/#учитель), который закладывает зерно всего Высшего.
