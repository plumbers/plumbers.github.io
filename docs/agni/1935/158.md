[Мир огненный 3 1935г](/agni/1935)

___158___
Для явлений [космических](/tag/#космос) [энергий](/tag/#[энергия](/tag/#энергия)) нужно особенно явить осторожность. Злоупотребление энергиями сказывается на всем утверждении [космических](/tag/#космос) [сил](/tag/#сила). Только сознательное и бережное отношение может устранить те страшные последствия. Вызванные [силы](/tag/#сила) из Тонкого Мира нуждаются в обуздании, которое лишь сильный [дух](/tag/#Дух) может проявить. Иначе эта необузданная сила явится утверждением [Космического](/tag/#космос) Хаоса. Когда приближаются огненные [сроки](/tag/#срок), нужно очень [знать](/tag/#познание) это, ибо велико будет явление вызываний.
