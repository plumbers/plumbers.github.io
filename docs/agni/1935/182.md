[Мир огненный 3 1935г](/agni/1935)

___182___
Да, да, да, велика [работа](/tag/#работа) огненных центров. Разрядители сфер есть самые мощные Служители [Космоса](/tag/#космос). Самые тонкие нити объединяют между собою этих великих Служителей [Космоса](/tag/#космос). Но эта [работа](/tag/#работа) тоже происходит только при огненном объединении. Огненное [равновесие](/tag/#равновесие) может спасти [планету](/tag/#планета). Только огненная мощь может в последний момент дать новую жизнь. Творчество объединенного [сердца](/tag/#сердце) даст спасение [планеты](/tag/#планета) и утвердит Новый Цикл. Потому Наше [Сердце](/tag/#сердце) напряжено в унисон стремительным током [Огня](/tag/#огонь) — так [сущность](/tag/#сущность) [жизни](/tag/#жизнь) являет свое Огненное Право. [Напряжение](/tag/#напряжение) велико, в Мире осуществляется великая Мировая Тайна. Я утверждаю великую [Космическую](/tag/#космос) [Истину](/tag/#истина). Тайна эта есть само Бытие.
