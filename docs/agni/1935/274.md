[Мир огненный 3 1935г](/agni/1935)

___274___
Потому [сердце](/tag/#сердце) пламенное может утверждать каждое явление. Ибо явленный магнит [сердца](/tag/#сердце) притягивает фохатические искры. Потому [сердце](/tag/#сердце), которое творит во имя [Космического](/tag/#космос) Права, имеет ту мощную силу и собирает фохатические искры и облекает ими те явления, которые напрягаются для творчества. Этот магнит слитого [сердца](/tag/#сердце) творит на всех планах. Потому Наше [Сердце](/tag/#сердце) творит так огненно. Есть [сердце](/tag/#сердце), которое может вместить каждую утвержденную степень [Огня](/tag/#огонь). Такое [сердце](/tag/#сердце) может утверждаться лишь [Космическим](/tag/#космос) Правом. [Сердце](/tag/#сердце), познавшее [Космическое](/tag/#космос) Право, имеет все [Огни](/tag/#огонь).
