[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___274___
Потому [сердце](../../../tags/#сердце) пламенное может утверждать каждое явление. Ибо явленный магнит [сердца](../../../tags/#сердце) притягивает фохатические искры. Потому [сердце](../../../tags/#сердце), которое творит во имя [Космического](../../../tags/#космос) Права, имеет ту мощную силу и собирает фохатические искры и облекает ими те явления, которые напрягаются для творчества. Этот магнит слитого [сердца](../../../tags/#сердце) творит на всех планах. Потому Наше [Сердце](../../../tags/#сердце) творит так огненно. Есть [сердце](../../../tags/#сердце), которое может вместить каждую утвержденную степень [Огня](../../../tags/#огонь). Такое [сердце](../../../tags/#сердце) может утверждаться лишь [Космическим](../../../tags/#космос) Правом. [Сердце](../../../tags/#сердце), познавшее [Космическое](../../../tags/#космос) Право, имеет все [Огни](../../../tags/#огонь).
