[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___146___

[Пространство](../../../tags/#пространство) дышит. [Пространство](../../../tags/#пространство) звучит и творит. О пространственных токах так же мало знают, как и о других высших Мирах. Огненная [сущность](../../../tags/#сущность), которая насыщает все живущее, есть [Космический](../../../tags/#космос) [Огонь](../../../tags/#огонь), исходящий из Недр [Космоса](../../../tags/#космос), идущий в беспредельном творческом явлении. Правильно сказано о чудесах [жизни](../../../tags/#жизнь). [Закон](../../../tags/#[закон](../../../tags/#закон)) [Космоса](../../../tags/#космос) есть огненное творчество. Оплодотворение [Космической](../../../tags/#космос) [Энергии](../../../tags/#энергия) есть [закон](../../../tags/#закон) [Космоса](../../../tags/#космос). Его вездесущность может являть свое [напряжение](../../../tags/#напряжение). Его вездесущность выражается во всей [жизни](../../../tags/#жизнь). Этот [Огонь](../../../tags/#огонь) [пространства](../../../tags/#пространство) оплодотворяет [мысль](../../../tags/#мысль) объединением тонких [энергий](../../../tags/#энергия). [Пространство](../../../tags/#пространство) заключает в себе тонкие формы для материализации. Нужно лишь пробудить в себе те [энергии](../../../tags/#энергия), которые могут объединиться для творчества. [Мысль](../../../tags/#мысль) и [устремление](../../../tags/#устремление) есть те предвестники, которые могут притянуть пространственное оплодотворение. В древние времена знали [значение](../../../tags/#значение) взываний к Высшим Существам. Потому мыслетворчество есть великое явление в [Космосе](../../../tags/#космос), ибо [Огонь](../../../tags/#огонь) [пространства](../../../tags/#пространство) принимает формы в духе, явленном на планете. Так объединение Миров утверждается жизненно.
