[Мир огненный 3 1935г](/agni/1935)

___146___
[Пространство](/tag/#пространство) дышит. [Пространство](/tag/#пространство) звучит и творит. О пространственных токах так же мало знают, как и о других высших Мирах. Огненная [сущность](/tag/#сущность), которая насыщает все живущее, есть [Космический](/tag/#космос) [Огонь](/tag/#огонь), исходящий из Недр [Космоса](/tag/#космос), идущий в беспредельном творческом явлении. Правильно сказано о чудесах [жизни](/tag/#жизнь). [Закон](/tag/#[закон](/tag/#закон)) [Космоса](/tag/#космос) есть огненное творчество. Оплодотворение [Космической](/tag/#космос) [Энергии](/tag/#энергия) есть [закон](/tag/#закон) [Космоса](/tag/#космос). Его вездесущность может являть свое [напряжение](/tag/#напряжение). Его вездесущность выражается во всей [жизни](/tag/#жизнь). Этот [Огонь](/tag/#огонь) [пространства](/tag/#пространство) оплодотворяет [мысль](/tag/#мысль) объединением тонких [энергий](/tag/#энергия). [Пространство](/tag/#пространство) заключает в себе тонкие формы для материализации. Нужно лишь пробудить в себе те [энергии](/tag/#энергия), которые могут объединиться для творчества. [Мысль](/tag/#мысль) и [устремление](/tag/#устремление) есть те предвестники, которые могут притянуть пространственное оплодотворение. В древние времена знали [значение](/tag/#значение) взываний к Высшим Существам. Потому мыслетворчество есть великое явление в [Космосе](/tag/#космос), ибо [Огонь](/tag/#огонь) [пространства](/tag/#пространство) принимает формы в духе, явленном на планете. Так объединение Миров утверждается жизненно.
