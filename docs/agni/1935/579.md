[Мир огненный 3 1935г](/agni/1935)

Живая Этика есть мост ко всем Мирам. Только в живом приложении создается прохождение неуязвленное. Ничто не поразит доспех огненный. Можно не тревожиться зарослями Метафизики, когда [дух](/tags/Дух) знает [путь](/tags/путь) живой [мысли](/tags/мысль). Только мера [добра](/tags/добро) явит [Огонь](/tags/огонь) Светлый. С таким светильником можно вступить ясно на великий мост. Только для [пути](/tags/путь) дальнего дается Живая Этика. Нужно любить ее, как путевое пособие.
