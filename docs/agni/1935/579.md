[Мир огненный 3 1935г](/agni/1935)

___579___
Живая Этика есть мост ко всем Мирам. Только в живом приложении создается прохождение неуязвленное. Ничто не поразит доспех огненный. Можно не тревожиться зарослями Метафизики, когда [дух](/tag/#Дух) знает [путь](/tag/#путь) живой [мысли](/tag/#мысль). Только мера [добра](/tag/#добро) явит [Огонь](/tag/#огонь) Светлый. С таким светильником можно вступить ясно на великий мост. Только для [пути](/tag/#путь) дальнего дается Живая Этика. Нужно любить ее, как путевое пособие.
