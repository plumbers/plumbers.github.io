[Мир огненный 3 1935г](https://127.0.0.1:4002/agni/1935)

___186___

Именно, воскресение [духа](../../../tags/#Дух) даст новую Эпоху. Что же может сравниться с мощью [духа](../../../tags/#Дух)? Нет другого рычага, который бы мог напрячь [огни](../../../tags/#огонь) центров. Каждое творчество, которое будет насыщаться воскресшим духом, может быть залогом великой Эпохи. Каждое строительство, которое будет зиждиться на воскресении [духа](../../../tags/#Дух), может быть залогом восхождения. [Духовное](../../../tags/#Дух) продвижение начнется, лишь когда придет [понимание](../../../tags/#понимание) обновления [духа](../../../tags/#Дух). Невозможно пребывать в старых искажениях. Невозможно создавать великое Царство [Духа](../../../tags/#Дух) без осознания утверждения чистого огненного [понимания](../../../tags/#понимание) подвига. Так лишь обновление [духа](../../../tags/#Дух) дает прочный фундамент для нового строительства. В нем [человечество](../../../tags/#человечество) найдет свое великое назначение и свое [место](../../../tags/#место) в [Космосе](../../../tags/#космос). [Истинно](../../../tags/#истина), воскресение [духа](../../../tags/#Дух) будет творчеством Новой Эпохи.
