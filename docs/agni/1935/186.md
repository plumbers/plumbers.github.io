[Мир огненный 3 1935г](/agni/1935)

___186___
Именно, воскресение [духа](/tag/#Дух) даст новую Эпоху. Что же может сравниться с мощью [духа](/tag/#Дух)? Нет другого рычага, который бы мог напрячь [огни](/tag/#огонь) центров. Каждое творчество, которое будет насыщаться воскресшим духом, может быть залогом великой Эпохи. Каждое строительство, которое будет зиждиться на воскресении [духа](/tag/#Дух), может быть залогом восхождения. [Духовное](/tag/#Дух) продвижение начнется, лишь когда придет [понимание](/tag/#понимание) обновления [духа](/tag/#Дух). Невозможно пребывать в старых искажениях. Невозможно создавать великое Царство [Духа](/tag/#Дух) без осознания утверждения чистого огненного [понимания](/tag/#понимание) подвига. Так лишь обновление [духа](/tag/#Дух) дает прочный фундамент для нового строительства. В нем [человечество](/tag/#человечество) найдет свое великое назначение и свое [место](/tag/#место) в [Космосе](/tag/#космос). [Истинно](/tag/#истина), воскресение [духа](/tag/#Дух) будет творчеством Новой Эпохи.
