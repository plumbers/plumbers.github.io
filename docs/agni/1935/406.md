[Мир огненный 3 1935г](/agni/1935)

Насыщение центров [Огнем](/tags/огонь) высшим напрягает психическую [энергию](/tags/[энергия](/tags/энергия)). Когда [Огонь](/tags/огонь) центров бушует, психическая [энергия](/tags/энергия) тоже находится в [состоянии](/tags/состояние) высшего [напряжения](/tags/напряжение). Уравновесие этих огненных [сил](/tags/сила) после трансмутации дает новое возжжение центров. Процесс нагнетения психической [энергии](/tags/энергия) идет соответственно с трансмутацией. Центры собирают в себе кристаллы психической [энергии](/tags/энергия), которые утверждают мощь трансмутации. Агни Йог являет мощное нагнетение [энергии](/tags/энергия), которая творит соответственно с явленным пространственным [Огнем](/tags/огонь). Сокровище психической [энергии](/tags/энергия) может создать панацею мощную. [Сознание](/tags/сознание), помогающее устремлять [дух](/tags/Дух) к трансмутации центров, являет огненное [действо](/tags/действие).
