[Мир огненный 3 1935г](/agni/1935)

___406___
Насыщение центров [Огнем](/tag/#огонь) высшим напрягает психическую [энергию](/tag/#[энергия](/tag/#энергия)). Когда [Огонь](/tag/#огонь) центров бушует, психическая [энергия](/tag/#энергия) тоже находится в [состоянии](/tag/#состояние) высшего [напряжения](/tag/#напряжение). Уравновесие этих огненных [сил](/tag/#сила) после трансмутации дает новое возжжение центров. Процесс нагнетения психической [энергии](/tag/#энергия) идет соответственно с трансмутацией. Центры собирают в себе кристаллы психической [энергии](/tag/#энергия), которые утверждают мощь трансмутации. Агни Йог являет мощное нагнетение [энергии](/tag/#энергия), которая творит соответственно с явленным пространственным [Огнем](/tag/#огонь). Сокровище психической [энергии](/tag/#энергия) может создать панацею мощную. [Сознание](/tag/#сознание), помогающее устремлять [дух](/tag/#Дух) к трансмутации центров, являет огненное [действо](/tag/#действие).
