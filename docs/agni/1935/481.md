[Мир огненный 3 1935г](/agni/1935)

___481___
Ничто не возвращается, но все стремится беспредельно. Счастье, именно, в [беспредельности](/tag/#беспредельность). Каждое ограничение есть уже оскорбление высшему творчеству. Ограничение есть темница, но полет в [Беспредельность](/tag/#беспредельность) уже творит лебединые крылья. Так недаром сложилось название Лебедя.
