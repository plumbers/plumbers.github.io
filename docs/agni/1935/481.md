[Мир огненный 3 1935г](/agni/1935)

Ничто не возвращается, но все стремится беспредельно. Счастье, именно, в [беспредельности](/tags/беспредельность). Каждое ограничение есть уже оскорбление высшему творчеству. Ограничение есть темница, но полет в [Беспредельность](/tags/беспредельность) уже творит лебединые крылья. Так недаром сложилось название Лебедя.
