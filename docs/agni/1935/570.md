[Мир огненный 3 1935г](/agni/1935)

Который бумеранг сильнее? Когда затронет самого [Учителя](/tags/учитель), тогда оружие возвратится ужасно. Потому [понятие](/tags/понятие) Гуру охранено так крепко. Когда явление Гуру затронуто, то оружие злобное возвращается смертельно. Это не есть наказание, но лишь самосуд. Потому будем осторожны с высшими понятиями, в них касаемся [Огня](/tags/огонь).
