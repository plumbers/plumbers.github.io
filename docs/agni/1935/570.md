[Мир огненный 3 1935г](/agni/1935)

___570___
Который бумеранг сильнее? Когда затронет самого [Учителя](/tag/#учитель), тогда оружие возвратится ужасно. Потому [понятие](/tag/#понятие) Гуру охранено так крепко. Когда явление Гуру затронуто, то оружие злобное возвращается смертельно. Это не есть наказание, но лишь самосуд. Потому будем осторожны с высшими понятиями, в них касаемся [Огня](/tag/#огонь).
