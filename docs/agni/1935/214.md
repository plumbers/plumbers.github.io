[Мир огненный 3 1935г](/agni/1935)

Циклы, которые закладываются [Космическим](/tags/космос) Магнитом, имеют в своем основании утверждение Высших [Сил](/tags/сила). Эти огненные Циклы являются основами планетной [жизни](/tags/жизнь). [Космический](/tags/космос) Магнит строит соизмеримо с пространственной мощью. [Дух](/tags/Дух) Строителя Цикла должен быть той Мощью, которая соответствует назначению Цикла. Весь синтез Цикла должен был насыщаться в Духе, явленном для [Космического](/tags/космос) Синтеза. При огненном смещении Цикла опять дается Огненное Начало для очищения [планеты](/tags/планета). Так мало духов, понимающих основы огненного Бытия! Так мало духов, понимающих Кто стоит у Руля! Красота Цикла может озарить лишь [сознание](/tags/сознание), которое может понять Мощь Начал. Руль [жизни](/tags/жизнь) [планеты](/tags/планета) и основы Бытия утверждаются Огненным Правом. Так на [пути](/tags/путь) к Миру Огненному явим [понимание](/tags/понимание) Основанию Циклов.
