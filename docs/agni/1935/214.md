[Мир огненный 3 1935г](/agni/1935)

___214___
Циклы, которые закладываются [Космическим](/tag/#космос) Магнитом, имеют в своем основании утверждение Высших [Сил](/tag/#сила). Эти огненные Циклы являются основами планетной [жизни](/tag/#жизнь). [Космический](/tag/#космос) Магнит строит соизмеримо с пространственной мощью. [Дух](/tag/#Дух) Строителя Цикла должен быть той Мощью, которая соответствует назначению Цикла. Весь синтез Цикла должен был насыщаться в Духе, явленном для [Космического](/tag/#космос) Синтеза. При огненном смещении Цикла опять дается Огненное Начало для очищения [планеты](/tag/#планета). Так мало духов, понимающих основы огненного Бытия! Так мало духов, понимающих Кто стоит у Руля! Красота Цикла может озарить лишь [сознание](/tag/#сознание), которое может понять Мощь Начал. Руль [жизни](/tag/#жизнь) [планеты](/tag/#планета) и основы Бытия утверждаются Огненным Правом. Так на [пути](/tag/#путь) к Миру Огненному явим [понимание](/tag/#понимание) Основанию Циклов.
