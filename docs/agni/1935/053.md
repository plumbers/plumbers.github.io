[Мир огненный 3 1935г](/agni/1935)

Правильно сказали о грубости и насколько бессильны тонкие [энергии](/tags/энергия) против грубости. Никакое построение не может стоять на столбах грубости, потому каждое явление, насыщенное грубостью, не будет прочным, регресс неминуем. Полный распад будет там, где червь грубости разъедает основу. Каждое [человеческое](/tags/человечество) деяние подвергается той же [опасности](/tags/опасность). Грубое [действие](/tags/действие) может быть покрыто тысячью вожделений, и не скрыть его от рекордов [пространства](/tags/пространство). Каждое Государство должно заботиться об искоренении этого ужаса. Каждая Община должна бороться с этим бичом. Никакое тесное общежитие не может иметь явленную грубость в своей среде. [Народ](/tags/народ), воспитанный на грубости, должен будет пройти огненную трансмутацию, и допустивший такое разложение будет кармически ответственен. Также и общинники, которые пребывают в грубости, должны будут пройти через особое очищение. Но, конечно, грубость есть ужасная зараза, которая развивает разложение в окружающем. Так Государство не может преуспевать, будучи рассадником микробов грубости. Также общинник не будет [истинным](/tags/истина) удачником, если грубость одержит [дух](/tags/Дух). Так запомним в строительстве на [пути](/tags/путь) к Миру Огненному.
