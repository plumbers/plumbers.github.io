Можно удивляться насколько люди даже из безобидных философий почерпают лишь злобу. Можно удивляться насколько низко [сознание](/tag/#сознание), которое может впитывать только скверну. Разве люди забыли, что каждая философия, прежде всего, запрещает злобу?
