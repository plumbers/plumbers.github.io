[Мир огненный 2 1934г](https://127.0.0.1:4002/agni/1934)

___285___

[Учение](../../../tags/#учение) следует читать в разных состояниях, но следствие будет неодинаково. При смятенном уме [Учение](../../../tags/#учение) даст успокоение, при горе — утешение, при сомнении — утверждение, но для впитывания действенности [Учения](../../../tags/#учение) нужно покрыть случайные чувствования проникновением в сокровище Иерархии. Не только для утешения дано [Учение](../../../tags/#учение), но для движения по лестнице восхождения. Конечно, при особых обстоятельствах Мира, особенно трудно постигательное углубление. Не раз уже [Мир](../../../tags/#мир) содрогался на границе механики и [духа](../../../tags/#Дух). Именно, теперь такое время, усугубленное натиском [сил](../../../tags/#сила) темных. Разнообразен базар материальных отбросов, прежде всего, нужно оценить их, чтобы предложить новые ценности. Так умение переменить оценку [сознания](../../../tags/#сознание) будет уже порогом [будущего](../../../tags/#будущее). Допущение уже есть признание, и много темных гостей допущено человечеством. Такие приглашения утяжеляют переходное [состояние](../../../tags/#состояние). Нужно просить [сердце](../../../tags/#сердце) поднять [голос](../../../tags/#голос) на обновление Мира.
