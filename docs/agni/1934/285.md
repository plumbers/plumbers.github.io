[Мир огненный 2 1934г](/agni/1934)

___285___
[Учение](/tag/#учение) следует читать в разных состояниях, но следствие будет неодинаково. При смятенном уме [Учение](/tag/#учение) даст успокоение, при горе — утешение, при сомнении — утверждение, но для впитывания действенности [Учения](/tag/#учение) нужно покрыть случайные чувствования проникновением в сокровище Иерархии. Не только для утешения дано [Учение](/tag/#учение), но для движения по лестнице восхождения. Конечно, при особых обстоятельствах Мира, особенно трудно постигательное углубление. Не раз уже [Мир](/tag/#мир) содрогался на границе механики и [духа](/tag/#Дух). Именно, теперь такое время, усугубленное натиском [сил](/tag/#сила) темных. Разнообразен базар материальных отбросов, прежде всего, нужно оценить их, чтобы предложить новые ценности. Так умение переменить оценку [сознания](/tag/#сознание) будет уже порогом [будущего](/tag/#будущее). Допущение уже есть признание, и много темных гостей допущено человечеством. Такие приглашения утяжеляют переходное [состояние](/tag/#состояние). Нужно просить [сердце](/tag/#сердце) поднять [голос](/tag/#голос) на обновление Мира.
