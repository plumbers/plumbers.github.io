[Мир огненный 2 1934г](/agni/1934)

___465___
Правильно [думаете](/tag/#дума) о благодарности. Лучшее выражение благодарности будет в осознании великого Поручения. [Служение](/tag/#служение) настолько велико, что каждый шаг уже составляет [подвиг](/tag/#подвиг). С каждым днем, с каждой мыслью уже делается нечто значительное. Сама внутренняя торжественность должна отвечать явлению великому. В этой торжественности также выражается благодарность. Торжественность есть один из лучших магнитов. Так будем мыслить от самого великого, ибо к этой мере прилепится остальное.
