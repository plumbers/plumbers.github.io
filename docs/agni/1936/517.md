[АУМ 1936г](https://127.0.0.1:4002/agni/1936)

___517___
Никто не должен считать, что развенчание чувств преподается, если они сопоставляются с [энергий](../../../tags/#[энергия](../../../tags/#энергия)). Кто-то вообразит, что невместно упоминать Высшее Собеседование наряду с [энергией](../../../tags/#[энергия](../../../tags/#энергия)). Кому-то [энергия](../../../tags/#энергия) есть нечто заключенное в машине, но ничтожны будут такие [земные](../../../tags/#Земля) [представления](../../../tags/#представление). Нужно тоже полюбить доверенную, явленную [энергию](../../../tags/#энергия). Доверенная [энергия](../../../tags/#энергия) есть капля из Высшей Чаши.   

Так без любви нет продвижения.
