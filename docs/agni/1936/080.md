[АУМ 1936г](/agni/1936)

Великая [Любовь](/tags/[любовь](/tags/любовь)) заложена в основание Высшего Мира. Ответит этому [качеству](/tags/качество) только такая же [любовь](/tags/любовь). Самое явленное почитание не достигает назначения без [любви](/tags/любовь). Какая же преданность будет без [любви](/tags/любовь)? Какая же огненность в иссушенном [сердце](/tags/сердце)? После уявления [любви](/tags/любовь) можно ждать соизмеримости с Высшим Миром. Каждый [предмет](/tags/предмет) изучается только при [любви](/tags/любовь). Каждая трудность побеждается силою [любви](/tags/любовь).   

Поистине, великая [любовь](/tags/любовь) лежит основанием Высшего Мира!
