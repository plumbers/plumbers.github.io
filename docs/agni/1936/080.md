[АУМ 1936г](/agni/1936)

___80___
Великая Любовь заложена в основание Высшего Мира. Ответит этому [качеству](/tag/#качество) только такая же любовь. Самое явленное почитание не достигает назначения без любви. Какая же преданность будет без любви? Какая же огненность в иссушенном [сердце](/tag/#сердце)? После уявления любви можно ждать соизмеримости с Высшим Миром. Каждый [предмет](/tag/#предмет) изучается только при любви. Каждая трудность побеждается силою любви.   

Поистине, великая любовь лежит основанием Высшего Мира!
