Великая Любовь заложена в основание Высшего Мира. Ответит этому [качеству](../../../tags/#качество) только такая же любовь. Самое явленное почитание не достигает назначения без любви. Какая же преданность будет без любви? Какая же огненность в иссушенном [сердце](../../../tags/#сердце)? После уявления любви можно ждать соизмеримости с Высшим Миром. Каждый [предмет](../../../tags/#предмет) изучается только при любви. Каждая трудность побеждается силою любви.   

Поистине, великая любовь лежит основанием Высшего Мира!
