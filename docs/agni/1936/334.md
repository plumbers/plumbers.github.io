[АУМ 1936г](/agni/1936)

Поручаю [Учение](/tags/учение) каждому, кто живет во всех Мирах. Не считайте такое определение неприложимым. Человек, именно, живет во всех Мирах. Каждый день он навещает Миры, но не может осознать этих мимолетных отсутствий. Лишь немногие [понимают](/tags/понимание) ощущение отсутствия. Не требуется долгого времени для безвременного [духа](/tags/Дух). Такие ощущения очень характерны для развитых сознаний.
