[АУМ 1936г](https://127.0.0.1:4002/agni/1936)

___580___
Постоянная готовность есть [качество](../../../tags/#качество), которое нужно вырабатывать. Готовность не есть нервный порыв, не есть временное [напряжение](../../../tags/#напряжение). Готовность есть [гармония](../../../tags/#гармония) центров, всегда открытых к восприятию и воздействию. Человек, полный [гармонии](../../../tags/#гармония), всегда получает и всегда дает. [Сущность](../../../tags/#сущность) его всегда укрепляется в непрерывном токе. Не бывает даяния без получения. Прервать такой ток есть смерть продвижения. Всезнающий будет и вседающим. Поймем эту [истину](../../../tags/#истина) широко, не ограничивая плотными условиями.   

Существует [закон](../../../tags/#закон), по которому получение не должно быть [понято](../../../tags/#понятие), как собственность. Осознание такого [понятия](../../../tags/#понятие) может происходить в [сердце](../../../tags/#сердце). Никакие лжеуверения не обманут [сердца](../../../tags/#сердце). Укрепление психической [энергии](../../../tags/#энергия) дает твердость.   

Постоянная готовность есть продукт здоровой психической [энергии](../../../tags/#энергия).
