[АУМ 1936г](/agni/1936)

___580___
Постоянная готовность есть [качество](/tag/#качество), которое нужно вырабатывать. Готовность не есть нервный порыв, не есть временное [напряжение](/tag/#напряжение). Готовность есть [гармония](/tag/#гармония) центров, всегда открытых к восприятию и воздействию. Человек, полный [гармонии](/tag/#гармония), всегда получает и всегда дает. [Сущность](/tag/#сущность) его всегда укрепляется в непрерывном токе. Не бывает даяния без получения. Прервать такой ток есть смерть продвижения. Всезнающий будет и вседающим. Поймем эту [истину](/tag/#истина) широко, не ограничивая плотными условиями.   

Существует [закон](/tag/#закон), по которому получение не должно быть [понято](/tag/#понятие), как собственность. Осознание такого [понятия](/tag/#понятие) может происходить в [сердце](/tag/#сердце). Никакие лжеуверения не обманут [сердца](/tag/#сердце). Укрепление психической [энергии](/tag/#энергия) дает твердость.   

Постоянная готовность есть продукт здоровой психической [энергии](/tag/#энергия).
