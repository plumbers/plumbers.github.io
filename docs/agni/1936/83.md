___83___

Чудеса не могут быть чем-то отвлеченным для [духа](../../../tags/#Дух), соединенного с Миром Высшим. Каждое необычное [земное](../../../tags/#Земля) явление есть частица самого Высшего Мира, иначе говоря, реальность. То же самое гармоничное звучание уже открывает сокровенные входы. Но замечайте самые малые знаки Мира Высшего. Из таких малых зерен вырастет древо, прочное и высочайшее.   

Нужно внимательно замечать все знаки. Не упустите немалые явления, которые вы считаете, в обмане плоти, не заслуживающими [внимания](../../../tags/#внимание) — плоть груба, только [сердце](../../../tags/#сердце) бьется во имя Высшего Мира. Аум!
