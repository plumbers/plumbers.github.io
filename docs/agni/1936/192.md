[АУМ 1936г](/agni/1936)

Да не подумает кто, что, зовя к Миру Высшему, Мы отрываем от [Земли](/tags/мысль) доброкачественными.   

Только зло может разделять миры; только невежество может расчленять явления; только непонимание может остерегать, что [земная](/tags/Земля) жизнь не есть часть прекрасного творчества, — потому направим всю науку, чтобы подойти к познанию праведному. Ничто не может отвратить [сердце](/tags/сердце), если в нем есть [чувство](/tags/чувство) преданности, [чувство](/tags/чувство) прекрасного.
