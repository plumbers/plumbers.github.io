[АУМ 1936г](/agni/1936)

___475___
Также называем ту же [энергию](/tag/#энергия) справедливостью. Если при воздействии [энергии](/tag/#энергия) можно определять различные свойства [людей](/tag/#люди), то, конечно, это будет [путем](/tag/#путь) справедливости. При опытах с психической [энергией](/tag/#энергия) можно убеждаться, насколько внешнее впечатление не соответствует внутреннему [состоянию](/tag/#состояние).   

Умение призвать на [помощь](/tag/#помощь) психическую [энергию](/tag/#энергия) будет [истинным](/tag/#истина) украшением судьи.
