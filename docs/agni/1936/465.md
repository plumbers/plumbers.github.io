[АУМ 1936г](/agni/1936)

Эктоплазма есть хранилище психической [энергии](/tags/энергия).   

Следует помнить, что, давая эктоплазму для случайных приходящих, медиум подвергается большой [опасности](/tags/опасность). Нельзя предоставлять такое ценное вещество для непрошеных посетителей. Тем ценнее Собеседования Высшие, они не истощают наших [сил](/tags/сила), иначе говоря, дают новый приток [силы](/tags/сила). Нужно понять, что психические исследования должны производиться целесообразно. Нельзя выпивать чью-то [сущность](/tags/сущность).
