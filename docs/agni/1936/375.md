[АУМ 1936г](/agni/1936)

[Энергия](/tags/энергия) может применяться решительно во всех случаях. Она может показать степень намагничивания [предметов](/tags/предмет) или воды. Она, как самый чуткий [аппарат](/tags/аппарат), может мгновенно улавливать колебания токов на дальнем расстоянии. Она может следить за [мыслями](/tags/мысль) каждой строки рукописи. Она есть показатель [качества](/tags/качество) излучения. Она в руках добрых есть орудие [добра](/tags/[добро](/tags/добро)).   

Конечно, многие, по счастью, не знают подступа к мощи. Только при улучшении [сознания](/tags/сознание) можно доверить психическую [энергию](/tags/энергия) для широкого пользования. Пусть это доброе время наступит скорей!
