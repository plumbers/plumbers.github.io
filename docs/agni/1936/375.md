[АУМ 1936г](/agni/1936)

___375___
[Энергия](/tag/#энергия) может применяться решительно во всех случаях. Она может показать степень намагничивания [предметов](/tag/#предмет) или воды. Она, как самый чуткий [аппарат](/tag/#аппарат), может мгновенно улавливать колебания токов на дальнем расстоянии. Она может следить за [мыслями](/tag/#мысль) каждой строки рукописи. Она есть показатель [качества](/tag/#качество) излучения. Она в руках добрых есть орудие [добра](/tag/#[добро](/tag/#добро)).   

Конечно, многие, по счастью, не знают подступа к мощи. Только при улучшении [сознания](/tag/#сознание) можно доверить психическую [энергию](/tag/#энергия) для широкого пользования. Пусть это доброе время наступит скорей!
