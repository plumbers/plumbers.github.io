[АУМ 1936г](/agni/1936)

[Духовное](/tags/Дух) начало предшествует каждому [действию](/tags/действие). Не может быть [действия](/tags/действие) телесного без предшествующего [духовного](/tags/Дух) соединения. Так каждый, отрицающий [духовное](/tags/Дух) начало, уже лишает смысла все свои [действия](/tags/действие). Не может продолжаться эволюция, если главный двигатель будет отринут. Черный [Век](/tags/сущность) Света.   

Когда говорю о Высшем Собеседовании, прежде всего, предлагаю понять реальность во всей [беспредельности](/tags/беспредельность).
