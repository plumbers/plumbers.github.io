[АУМ 1936г](https://127.0.0.1:4002/agni/1936)

___272___
Напрасно некто утверждает, что [Мир](../../../tags/#мир) Невидимый не существует — такая ложь будет равносильна отрицанию [мысли](../../../tags/#мысль). [Мысль](../../../tags/#мысль) тоже невидима, но лишь невежество отрицает мыслительный процесс. Так можно начать отвергать все [энергии](../../../tags/#энергия), ибо они невидимы. К тому же, разве [Мир](../../../tags/#мир) Невидимый для всех незрим? Пусть отрицатели не судят по себе. Суждение по себе есть рассадник самости.
