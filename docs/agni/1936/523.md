[АУМ 1936г](/agni/1936)

___523___
К тому же, люди должны пересмотреть словесные [понятия](/tag/#[понятие](/tag/#понятие)). Сегодня уместно сказать о торжественности, но многие ошибочно поймут такое прекрасное [понятие](/tag/#понятие). Для многих торжественность есть праздничное безделье, есть безответственное хождение и произнесение отживших [слов](/tag/#слово). На самом [деле](/tag/#дел), торжественность есть возвышенное приношение всех лучших чувств, есть [напряжение](/tag/#напряжение) всех превосходных [энергий](/tag/#энергия), прикосновение к следующим Вратам.
