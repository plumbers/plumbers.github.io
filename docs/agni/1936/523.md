___523___

К тому же, люди должны пересмотреть словесные [понятия](../../../tags/#[понятие](../../../tags/#понятие)). Сегодня уместно сказать о торжественности, но многие ошибочно поймут такое прекрасное [понятие](../../../tags/#понятие). Для многих торжественность есть праздничное безделье, есть безответственное хождение и произнесение отживших [слов](../../../tags/#слово). На самом [деле](../../../tags/#дел), торжественность есть возвышенное приношение всех лучших чувств, есть [напряжение](../../../tags/#напряжение) всех превосходных [энергий](../../../tags/#энергия), прикосновение к следующим Вратам.
