[АУМ 1936г](/agni/1936)

___11___
Могу радоваться, когда вижу воинов бодрыми. [Путей](/tag/#путь) много, и преследователи не угонятся. Кроме того, каждая [битва](/tag/#битва) с тьмою есть [действие](/tag/#действие) достойное. Каждое рассеяние тьмы есть долг [человека](/tag/#человек). Герой вызывает трубным звуком дракона, чтобы поразить его. Пока змей под землею, не будут у очага спокойны люди. Каждое истребление скверны уже будет строительством [будущего](/tag/#будущее). Не может смущаться герой.
