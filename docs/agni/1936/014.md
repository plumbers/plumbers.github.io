[АУМ 1936г](/agni/1936)

___14___
Живая [мысль](/tag/#мысль) из [Беспредельности](/tag/#беспредельность) есть уже утверждение [человека](/tag/#человек) как одухотворенного существа, как посланца, как стража светлого. Немногие поймут чудесное [значение](/tag/#значение) живой [мысли](/tag/#мысль) пространственной. Разве не расцветет [мир](/tag/#мир) для [сознания](/tag/#сознание), усвоившего красоту живой [мысли](/tag/#мысль)?   

Утверждаю, что из [Беспредельности](/tag/#беспредельность) льется [мысль](/tag/#мысль) на досягаемом выражении.
