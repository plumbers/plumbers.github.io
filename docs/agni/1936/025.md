[АУМ 1936г](/agni/1936)

К чему говорить — Аум, если можно сказать — молитва? В [сущности](/tags/сущность) это то же самое, только по древности и утончению созвучие Аум будет сильнее по [вибрации](/tags/[вибрация](/tags/вибрация)). Пусть созвучие около высшего [Понятия](/tags/понятие) будет обдумано. [Слово](/tags/слово) есть [вибрация](/tags/вибрация), такие созвучия нужны для [гармонии](/tags/гармония) [пространства](/tags/пространство).   

Подвижники молятся не о себе.
