[АУМ 1936г](/agni/1936)

___25___
К чему говорить — Аум, если можно сказать — молитва? В [сущности](/tag/#сущность) это то же самое, только по древности и утончению созвучие Аум будет сильнее по [вибрации](/tag/#[вибрация](/tag/#вибрация)). Пусть созвучие около высшего [Понятия](/tag/#понятие) будет обдумано. [Слово](/tag/#слово) есть [вибрация](/tag/#вибрация), такие созвучия нужны для [гармонии](/tag/#гармония) [пространства](/tag/#пространство).   

Подвижники молятся не о себе.
