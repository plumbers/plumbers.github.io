[АУМ 1936г](/agni/1936)

Наблюдения следует вести не только над согласованными привходящими, но также и над разъединяющими появлениями. Опыт ценен разносторонний. Невозможно предрешить при начале исследования, какие именно ингредиенты потребуются для усиления следствия.   

Можно призвать [сотрудничество](/tags/сотрудничество) самых неожиданных [предметов](/tags/предмет), ибо свойства тончайших [энергий](/tags/возможности) нисколько не нарушает научности исследования. Можно применить индивидуальные методы и такие новые проявления мужественно принять.   

Никто не может указать, где кончается мощь [человека](/tags/человек). При этом не сверхчеловек, но, именно, самый здоровый человек может окрылиться счастливым достижением. В каждом обиходе может быть изучаема психическая [энергия](/tags/энергия). Не нужно особых дорогих лабораторий, чтобы воспитывать [сознание](/tags/сознание).   

Каждый [век](/tags/век) несет свою весть [человечеству](/tags/человечество). Психическая [энергия](/tags/энергия) имеет назначение помочь [человечеству](/tags/человечество) среди нерешимых для него проблем.
