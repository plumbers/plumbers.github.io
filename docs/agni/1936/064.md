[АУМ 1936г](/agni/1936)

Ужасно видеть явление безумия, когда злоба хочет стереть с лица [Земли](/tags/Земля) все разумное; подобно губительному вихрю [действует](/tags/действие) злоба. Только связь с Высшим Миром может дать [равновесие](/tags/равновесие).
