[Сердце 1932г](https://127.0.0.1:4002/agni/1932)

___195___

Не будем удивляться, если сейчас [сердца](../../../tags/#сердце) отягощены перекрёстными токами необычайного [напряжения](../../../tags/#напряжение). Токи наций, токи мира, наконец токи сильных личностей перекрещиваются; но через них идут токи Тонкого Мира, где сейчас столько [действий](../../../tags/#действие)! [События](../../../tags/#событие) только ещё формируются; можно [представить](../../../tags/#представление), насколько [мировое](../../../tags/#мир) [сознание](../../../tags/#сознание) [действует](../../../tags/#действие) на слабые мозги!
