[Сердце 1932г](/agni/1932)

___195___
Не будем удивляться, если сейчас [сердца](/tag/#сердце) отягощены перекрёстными токами необычайного [напряжения](/tag/#напряжение). Токи наций, токи мира, наконец токи сильных личностей перекрещиваются; но через них идут токи Тонкого Мира, где сейчас столько [действий](/tag/#действие)! [События](/tag/#событие) только ещё формируются; можно [представить](/tag/#представление), насколько [мировое](/tag/#мир) [сознание](/tag/#сознание) [действует](/tag/#действие) на слабые мозги!
