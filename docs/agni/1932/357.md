[Сердце 1932г](/agni/1932)

Для восстановления [сердца](/tags/сердце), при разных причинах, употребляем [вибрации](/tags/[вибрация](/tags/вибрация)). Если Махаван может отвечать [земным](/tags/Земля) причинам, то [вибрация](/tags/вибрация) Серебряного Моста нужна при пертурбациях Тонкого Мира.
