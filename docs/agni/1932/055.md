[Сердце 1932г](/agni/1932)

Пространственное [мышление](/tags/мышление) вовсе не легко для большинства. Прежде всего, для него нужно сохранить личность, но освободиться от эгоизма. Многим такое противоположение покажется вообще нелепым; для них эгоизм и есть личность. Явление мощной личности, преданной Общему Благу, многим невообразимо, но без личности [мышление](/tags/мышление) не будет потенциально. При эгоизме [мышление](/tags/мышление) прибавит ещё одну порцию яда к заражённой ауре [планеты](/tags/планета). Также трудно многим осознать, что вещество [мысли](/tags/мысль)?
