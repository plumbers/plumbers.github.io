[Сердце 1932г](/agni/1932)

Нужно ясно вообразить непрестанное излучение [сердца](/tags/[сердце](/tags/сердце)). Нужно понять, почему [сердцу](/tags/[сердце](/tags/сердце)) [добра](/tags/добро) так болезненно присутствие сердец зла. Ни улыбка, ни насильственная ужимка зла не скроют сердечное излучение. Утверждение [добра](/tags/добро) в [сердце](/tags/сердце) вовсе не исключает справедливого негодования, но раздражение есть область зла. Только [устремление](/tags/устремление) к Иерархии может утвердить границу между многими чувствованиями.
