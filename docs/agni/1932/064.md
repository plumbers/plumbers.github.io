[Сердце 1932г](/agni/1932)

___64___
Нужно ясно вообразить непрестанное излучение [сердца](/tag/#[сердце](/tag/#сердце)). Нужно понять, почему [сердцу](/tag/#[сердце](/tag/#сердце)) [добра](/tag/#добро) так болезненно присутствие сердец зла. Ни улыбка, ни насильственная ужимка зла не скроют сердечное излучение. Утверждение [добра](/tag/#добро) в [сердце](/tag/#сердце) вовсе не исключает справедливого негодования, но раздражение есть область зла. Только [устремление](/tag/#устремление) к Иерархии может утвердить границу между многими чувствованиями.
