[Сердце 1932г](/agni/1932)

___128___
Корень [представления](/tag/#[представление](/tag/#представление)) — глубокая основа Бытия. Без [представления](/tag/#[представление](/tag/#представление)) не может быть [устремления](/tag/#устремление) [познания](/tag/#познание) и творения. Как может [дух](/tag/#Дух) творить, если он не убеждён представлением? Как может говорить о Высшем Начале, если в Бытие не будет заложено [представление](/tag/#представление)? По этому мерилу различаются ценности [духа](/tag/#Дух). Явление это без [представления](/tag/#[представление](/tag/#представление)) окажется танцем скелета. Но, как видите, сейчас нужен бальзам для гниющих частей [человечества](/tag/#человечество).
