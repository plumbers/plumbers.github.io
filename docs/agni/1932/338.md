[Сердце 1932г](/agni/1932)

Нужно иметь много [мужества](/tags/мужество), чтобы среди разрушения слагать [силы](/tags/сила) [сердца](/tags/сердце). Если бы не [знать](/tags/познание) необходимость этих преображений для всех миров, можно бы подумать о тщетности сердечных достижений. Но, по счастью, это нужно для всего сущего. Потому даже в пыли разрушений следует слагать твердыни [сердца](/tags/сердце).
