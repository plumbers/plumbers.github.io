[Сердце 1932г](/agni/1932)

___338___
Нужно иметь много [мужества](/tag/#мужество), чтобы среди разрушения слагать [силы](/tag/#сила) [сердца](/tag/#сердце). Если бы не [знать](/tag/#познание) необходимость этих преображений для всех миров, можно бы подумать о тщетности сердечных достижений. Но, по счастью, это нужно для всего сущего. Потому даже в пыли разрушений следует слагать твердыни [сердца](/tag/#сердце).
