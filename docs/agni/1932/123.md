[Сердце 1932г](/agni/1932)

___123___
Хвалите врачей, которые в начале каждой [болезни](/tag/#[болезнь](/tag/#болезнь)) дают сильное тоническое средство. Поздно гнаться за болезнью, когда уже обессилело [сердце](/tag/#сердце). Усмотреть зарождение [болезни](/tag/#[болезнь](/tag/#болезнь)) и влить новые [силы](/tag/#сила) для борьбы с нею есть задача врача, потому Мы, прежде всего, обращаем [внимание](/tag/#внимание) ваше на мускус. Но не достаточно для всех драгоценного вещества, потому ещё раз обращаем [внимание](/tag/#внимание) на растения, входящие в пищу этих животных. Конечно, этот состав будет слабее, но всё-таки он даст оздоровляющее вещество, которое можно применять широко. Так можно будет избежать главного врага [человечества](/tag/#человечество) — всех наркотиков. Не очень трудно находить составы пищи животных, к тому же можно найти способ обходиться без убийства.
