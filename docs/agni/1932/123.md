[Сердце 1932г](/agni/1932)

Хвалите врачей, которые в начале каждой [болезни](/tags/[болезнь](/tags/болезнь)) дают сильное тоническое средство. Поздно гнаться за болезнью, когда уже обессилело [сердце](/tags/сердце). Усмотреть зарождение [болезни](/tags/[болезнь](/tags/болезнь)) и влить новые [силы](/tags/сила) для борьбы с нею есть задача врача, потому Мы, прежде всего, обращаем [внимание](/tags/внимание) ваше на мускус. Но не достаточно для всех драгоценного вещества, потому ещё раз обращаем [внимание](/tags/внимание) на растения, входящие в пищу этих животных. Конечно, этот состав будет слабее, но всё-таки он даст оздоровляющее вещество, которое можно применять широко. Так можно будет избежать главного врага [человечества](/tags/человечество) — всех наркотиков. Не очень трудно находить составы пищи животных, к тому же можно найти способ обходиться без убийства.
