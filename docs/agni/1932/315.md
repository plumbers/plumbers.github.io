[Сердце 1932г](https://127.0.0.1:4002/agni/1932)

___315___

Успокоение есть уравновесие [напряжения](../../../tags/#напряжение). Объединение [сознания](../../../tags/#сознание) есть, прежде всего, сохранение [энергии](../../../tags/#энергия). Это важное правило обычно забывается. Когда вместо ограниченной физиологии введётся психофизиология, тогда можно будет каждому понять, какое [значение](../../../tags/#значение) имеет экономия [энергии](../../../tags/#энергия).
