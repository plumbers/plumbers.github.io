[Сердце 1932г](/agni/1932)

___315___
Успокоение есть уравновесие [напряжения](/tag/#напряжение). Объединение [сознания](/tag/#сознание) есть, прежде всего, сохранение [энергии](/tag/#энергия). Это важное правило обычно забывается. Когда вместо ограниченной физиологии введётся психофизиология, тогда можно будет каждому понять, какое [значение](/tag/#значение) имеет экономия [энергии](/tag/#энергия).
