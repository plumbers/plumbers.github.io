[Сердце 1932г](https://127.0.0.1:4002/agni/1932)

___470___

Пусть признают [сущность](../../../tags/#сущность) [сердца](../../../tags/#сердце) как субстрат негасимый. Термин не важен, но [сущность](../../../tags/#сущность) [сердца](../../../tags/#сердце) очевидна. Так нужно привыкать к непреложным понятиям, близким всему [человечеству](../../../tags/#человечество). Пострадавшие узнают, почему с незапамятных времён убит Осирис и раскинут по всему миру. Без убиения не могли бы быть посланы вестники по свету. Так страдание с одной точки зрения — будет лишь распространением с другой. Также знаменательно чтение трудов, заключённых во многих [книгах](../../../tags/#книга). Изучающий не норовит усвоить [Учение](../../../tags/#учение) при одинаковом настроении, и, таким [образом](../../../tags/#образ), рождается множество точек зрения. Потому [мудро](../../../tags/#мудрость) в каждой книге хотя бы кратко коснуться ранее изложенного, чтобы дать [возможность](../../../tags/#возможности) воспринять при одинаковости настроения. Настроение есть рождение точек зрения.
