[Сердце 1932г](/agni/1932)

___519___
Действительно, отложение впечатления в третий глаз есть основа творчества. Не только старые буддисты, но незапамятные Заветы требовали воспитания наблюдательности. [Сердце](/tag/#сердце), лишённое сокровища наблюдательности, затрачивает массу [энергии](/tag/#энергия) там, где нужно проявить великую бережность. [Учитель](/tag/#учитель) должен развивать наблюдательность на самых прекрасных предметах. Особенно непростительно в [человеке](/tag/#человек) скользкий взгляд, ничего не замечающий, ничего не дающий. Химизм взгляда разве не будет благодетельным заданием для истинного учёного?
