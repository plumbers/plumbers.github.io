[Сердце 1932г](/agni/1932)

[Учитель](/tags/учитель) посылает [понимание](/tags/понимание) [мировых](/tags/мир) [событий](/tags/событие). Без Иерархии хаос [событий](/tags/событие) обращается в лохматые клубы. Так верно заметили, что настоящее время не может уложиться в систему без очищения [сознания](/tags/сознание). Утверждение узловых движений [народов](/tags/народ) может быть сделано с Горы.
