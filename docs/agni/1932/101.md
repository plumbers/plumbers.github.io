[Сердце 1932г](/agni/1932)

___101___
Нужно соизмерять [силы](/tag/#сила), когда их требуют во всех направлениях. Можно понять [значение](/tag/#значение) [битвы](/tag/#битва), когда утомлённые воители возвращаются для новой [битвы](/tag/#битва). [Истинно](/tag/#истина), первая задача йога в распределении и экономии [сил](/tag/#сила). Недаром щедрый сердцем не будет напрасно махать руками. Ныне особенно нужна бережливость, ибо не надо лишними тратами напрягать [энергию](/tag/#энергия). Нужно быть на дозоре.
