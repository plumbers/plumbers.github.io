[Сердце 1932г](/agni/1932)

Нужно соизмерять [силы](/tags/сила), когда их требуют во всех направлениях. Можно понять [значение](/tags/значение) [битвы](/tags/битва), когда утомлённые воители возвращаются для новой [битвы](/tags/битва). [Истинно](/tags/истина), первая задача йога в распределении и экономии [сил](/tags/сила). Недаром щедрый сердцем не будет напрасно махать руками. Ныне особенно нужна бережливость, ибо не надо лишними тратами напрягать [энергию](/tags/энергия). Нужно быть на дозоре.
