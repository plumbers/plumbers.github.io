[Братство 1937г](/agni/1937)

___121___
Следует уметь преодолевать кажущиеся противоречия. С одной стороны, нужно воспитывать добросердечие, с другой, нужно постичь суровость. Для многих такая задача совершенно неразрешима, только [сердце](/tag/#сердце) может подсказать, когда оба [качества](/tag/#качество) не будут противоречить друг другу. [Сердце](/tag/#сердце) укажет, когда нужно броситься на [помощь](/tag/#помощь) ближнему. Оно же прикажет, когда пресечь безумие ярого животного. Нельзя выразить в слове закона, когда является необходимость того или иного [действия](/tag/#действие). Неписаны законы [сердца](/tag/#сердце), но лишь в нем живет справедливость, ибо [сердце](/tag/#сердце) есть мост миров.   

Где весы самоотвержения? Где судья подвига? Где мера долга? Меч умения может блеснуть по велению [сердца](/tag/#сердце). Для [сердца](/tag/#сердце) не будет противоречия.
