[Братство 1937г](https://127.0.0.1:4002/agni/1937)

___121___
Следует уметь преодолевать кажущиеся противоречия. С одной стороны, нужно воспитывать добросердечие, с другой, нужно постичь суровость. Для многих такая задача совершенно неразрешима, только [сердце](../../../tags/#сердце) может подсказать, когда оба [качества](../../../tags/#качество) не будут противоречить друг другу. [Сердце](../../../tags/#сердце) укажет, когда нужно броситься на [помощь](../../../tags/#помощь) ближнему. Оно же прикажет, когда пресечь безумие ярого животного. Нельзя выразить в слове закона, когда является необходимость того или иного [действия](../../../tags/#действие). Неписаны законы [сердца](../../../tags/#сердце), но лишь в нем живет справедливость, ибо [сердце](../../../tags/#сердце) есть мост миров.   

Где весы самоотвержения? Где судья подвига? Где мера долга? Меч умения может блеснуть по велению [сердца](../../../tags/#сердце). Для [сердца](../../../tags/#сердце) не будет противоречия.
