[Братство 1937г](/agni/1937)

Немыслимо не чувствовать [напряжения](/tags/напряжение) [космических](/tags/космос) токов, которые поглощают психическую [энергию](/tags/энергия). Может быть некоторая сонливость, может быть как бы рассеянность, может быть невольное раздражение — поучительно наблюдать такие признаки, которые сопровождают поглощение [энергии](/tags/энергия). Люди склонны приписывать их своему недомоганию, но не забудем внешние причины.
