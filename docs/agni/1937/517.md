[Братство 1937г](/agni/1937)

___517___
Немыслимо не чувствовать [напряжения](/tag/#напряжение) [космических](/tag/#космос) токов, которые поглощают психическую [энергию](/tag/#энергия). Может быть некоторая сонливость, может быть как бы рассеянность, может быть невольное раздражение — поучительно наблюдать такие признаки, которые сопровождают поглощение [энергии](/tag/#энергия). Люди склонны приписывать их своему недомоганию, но не забудем внешние причины.
