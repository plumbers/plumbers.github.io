[Братство 1937г](/agni/1937)

Всякое научное [познание](/tags/познание) прекрасно тем, что не содержит конечного тупика. Безысходность не знакома познавателю. Он может черпать постоянно, разрабатывая новые отрасли познавания. Для явлений подготовки к Братству такая [беспредельность](/tags/беспредельность) [познания](/tags/познание) есть лучшая ступень. Не очень легко познается такая [беспредельность](/tags/беспредельность), но для знающего продвижение эволюции она будет естественным и единственным [путем](/tags/путь). Только не дайте зачерстветь [сердцу](/tags/сердце). Без него не может строиться Братство.
