[Братство 1937г](https://127.0.0.1:4002/agni/1937)

___11___
Где-то запрещаются гомеопатические средства, также кто-то желает лечить [людей](../../../tags/#люди) лишь своим способом. Ограничено [мышление](../../../tags/#мышление) запрещающее. Невозможно установить один способ лечения. Следует помнить, что все лекарства являются лишь средствами вспомогательными. Без всеначальной [энергии](../../../tags/#энергия) никакое лекарство не окажет должного [действия](../../../tags/#действие).   

Нельзя [делить](../../../tags/#дел) врачей на аллопатов и гомеопатов, каждый применяет лучший метод индивидуально. Также врач знает основную [энергию](../../../tags/#энергия), которая будет фактором скорейшего выздоровления.
