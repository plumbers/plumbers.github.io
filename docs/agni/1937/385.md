[Братство 1937г](/agni/1937)

Рассудок есть водитель недоразумения. Рассудочное [мышление](/tags/мышление) осуждается, но и безрассудочные [действия](/tags/действие) осуждены. Значит, есть какая-то сила, которая должна дополнить деятельность рассудка. [Сердце](/tags/сердце) должно быть верховным судьей. Оно, как совесть [народов](/tags/народ), будет вносить [равновесие](/tags/равновесие). Рассудок не есть [равновесие](/tags/равновесие).
