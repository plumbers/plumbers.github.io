[Братство 1937г](/agni/1937)

___385___
Рассудок есть водитель недоразумения. Рассудочное [мышление](/tag/#мышление) осуждается, но и безрассудочные [действия](/tag/#действие) осуждены. Значит, есть какая-то сила, которая должна дополнить деятельность рассудка. [Сердце](/tag/#сердце) должно быть верховным судьей. Оно, как совесть [народов](/tag/#народ), будет вносить [равновесие](/tag/#равновесие). Рассудок не есть [равновесие](/tag/#равновесие).
