[Братство 1937г](/agni/1937)

[Опыты](/tags/опыт) над психической [энергией](/tags/энергия) покажут насколько такая симфония расширяет благодетельный круг. Опытные наблюдатели легко поймут соотношение [качеств](/tags/качество) к психической [энергии](/tags/энергия), но для невежд такое сопоставление будет непонятно.   

Для долгого [пути](/tags/путь) соберем [возможно](/tags/возможности) больше [качеств](/tags/качество). Пусть каждое из них будет лучшей степени!
