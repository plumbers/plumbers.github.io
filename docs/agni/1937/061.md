[Братство 1937г](/agni/1937)

___61___
[Опыты](/tag/#опыт) над психической [энергией](/tag/#энергия) покажут насколько такая симфония расширяет благодетельный круг. Опытные наблюдатели легко поймут соотношение [качеств](/tag/#качество) к психической [энергии](/tag/#энергия), но для невежд такое сопоставление будет непонятно.   

Для долгого [пути](/tag/#путь) соберем [возможно](/tag/#возможности) больше [качеств](/tag/#качество). Пусть каждое из них будет лучшей степени!
