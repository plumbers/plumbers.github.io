[Братство 1937г](/agni/1937)

«Преследователи ярые, куда гоните? Сами не [зная](/tags/познание), приближаете к светлому Прибежищу». Эта древняя песнь может быть повторена во всех веках. Можно на всех наречиях подтвердить такую [истину](/tags/истина), потому лучше быть гонимым, нежели гонителем.
