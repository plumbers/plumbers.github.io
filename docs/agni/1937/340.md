[Братство 1937г](/agni/1937)

___340___
«Преследователи ярые, куда гоните? Сами не [зная](/tag/#познание), приближаете к светлому Прибежищу». Эта древняя песнь может быть повторена во всех веках. Можно на всех наречиях подтвердить такую [истину](/tag/#истина), потому лучше быть гонимым, нежели гонителем.
