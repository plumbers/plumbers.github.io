[Братство 1937г](/agni/1937)

___135___
Потемки [духа](/tag/#Дух) порождаются самими людьми. Наследство Тонкого Мира остается не действительнее сновидения. Оно даже встречает враждебность рассудка. Не принимает рассудок явлений Высшего Мира. Особенно тяжко для него огненное сияние.
