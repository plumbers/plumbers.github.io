[Братство 1937г](/agni/1937)

___570___
[Свободная](/tag/#свобода) [воля](/tag/#воля) есть торжественное напутствие путнику. Невозможно не дать перед дальними [путями](/tag/#путь) ценного дара [свободной](/tag/#свобода) [воли](/tag/#воля). Каждый может действовать по способности, не будет он стеснен. Но [мудрый](/tag/#мудрость) осознает, какова ответственность за пользование сокровищем [свободной](/tag/#свобода) [воли](/tag/#воля). Как бы дан кошель, полный золота. Можно его расходовать по усмотрению, но придется дать отчет, и Братство научит, чтобы порученное сокровище не было потрачено без пользы.
