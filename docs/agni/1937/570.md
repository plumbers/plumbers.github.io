[Братство 1937г](/agni/1937)

[Свободная](/tags/свобода) [воля](/tags/воля) есть торжественное напутствие путнику. Невозможно не дать перед дальними [путями](/tags/путь) ценного дара [свободной](/tags/свобода) [воли](/tags/воля). Каждый может действовать по способности, не будет он стеснен. Но [мудрый](/tags/мудрость) осознает, какова ответственность за пользование сокровищем [свободной](/tags/свобода) [воли](/tags/воля). Как бы дан кошель, полный золота. Можно его расходовать по усмотрению, но придется дать отчет, и Братство научит, чтобы порученное сокровище не было потрачено без пользы.
