[Братство 1937г](/agni/1937)

Терпимость есть одно из [условий](/tags/условие) наблюдательности. [Истинная](/tags/истина) наблюдательность есть основа [познания](/tags/познание). Человек нетерпимый не может составить справедливого [представления](/tags/представление) о [вещах](/tags/вещь). Он лишает себя наблюдательности и теряет прозорливость. Какое же познавание может родиться из самости, отвергающей действительность? Много примеров, когда великие [Истины](/tags/истина) подвергались искажению, вследствие нетерпимости. Можно сказать, что нетерпимость есть невежество, но такое определение будет слишком мягко. Нетерпимость есть зло. Не может быть доброй нетерпимости. Она, непременно, содержит в себе и ложь, ибо скрывает правду. Только весьма неумные могут легкомысленно не считать нетерпимость чем-то недостойным.
